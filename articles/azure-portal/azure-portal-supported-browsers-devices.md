# <a name="supported-devices"></a>Dispositivos compatibles

El [portal Azure](https://portal.azure.com) es una consola basada en web y se ejecuta en el Explorador de todos los equipos de escritorio modernos y dispositivos de tableta gráfica. Si necesita administrar recursos de Azure desde un dispositivo móvil, pruebe el [aplicación móvil de Azure](https://azure.microsoft.com/features/azure-portal/mobile-app/). Está disponible para iOS y Android.

## <a name="recommended-browsers"></a>Exploradores recomendados

Se recomienda usar el explorador más actualizado compatible con el sistema operativo. Se admiten los siguientes exploradores:

* Microsoft Edge (versión más recientes)
* Internet Explorer 11
* Safari (versión más reciente, solo Mac)
* Chrome (versión más reciente)
* Firefox (versión más reciente)
