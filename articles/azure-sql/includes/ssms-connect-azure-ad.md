> [!NOTE]
> A partir de diciembre de 2021, las versiones de SSMS anteriores a la 18.6 ya no llevarán a cabo la autenticación por medio de Azure Active Directory con MFA.
>
> Para continuar utilizando la autenticación de Azure Active Directory con MFA, necesitará [SSMS 18.6 o versiones posteriores](/sql/ssms/download-sql-server-management-studio-ssms).
