# <a name="how-to-delete-an-invitation-to-a-recipient-in-azure-data-share"></a>Eliminación de una invitación para un destinatario en Azure Data Share

En este artículo se explica cómo eliminar una invitación para un destinatario antes de que la acepte. Esto impedirá a dicho destinatario crear una suscripción a un recurso compartido y nunca recibirá datos compartidos. Si elimina la invitación para un consumidor que ya ha creado una suscripción a un recurso compartido, la eliminación de dicha invitación no tendrá ningún efecto.

## <a name="navigate-to-a-sent-data-share"></a>Desplazamiento a un recurso compartido de datos enviado

En Azure Data Share, vaya a su recurso compartido enviado y seleccione la pestaña **Invitaciones**. Aquí se enumeran todos los destinatarios de invitaciones a este recurso compartido de datos. Active las casillas situadas junto a los destinatarios cuyas invitaciones desea eliminar y haga clic en **Eliminar**.
