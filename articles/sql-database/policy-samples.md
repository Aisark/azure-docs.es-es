Esta página es un índice de las definiciones de directivas integradas de [Azure Policy](../governance/policy/overview.md) para Azure SQL Database. Puede encontrar elementos integrados adicionales de Azure Policy para otros servicios en [Definiciones de elementos integrados de Azure Policy](../governance/policy/samples/built-in-policies.md).
