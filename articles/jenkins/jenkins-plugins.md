# <a name="jenkins-plugins-for-azure"></a>Complementos de Jenkins para Azure

Los siguientes complementos de Jenkins admiten varias características para su uso con Azure.
  
| Complemento de Jenkins | DESCRIPCIÓN                                   |
|------------------------------------------------------------------------------|
| [Complemento de Azure App Service](https://plugins.jenkins.io/azure-app-service)     | Un complemento de Jenkins para implementar un servicio de aplicación de Azure (actualmente admite solo la aplicación Web). | 
| [Complemento de Azure AD](https://plugins.jenkins.io/azure-ad)                       | Un Jenkins Plugin que admita la autenticación y autorización a través de Azure Active Directory. | 
| [Complemento de Azure Container Agents](https://plugins.jenkins.io/azure-container-agents) | Complemento de agentes de contenedor de Azure puede ayudarle a ejecutar un contenedor como un agente en Jenkins | 
| [Complemento de Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Un Jenkins Plugin para implementar configuraciones de Azure Container Service (AKS). | 
| [Complemento de Azure Credential](https://plugins.jenkins.io/azure-credentials)      | Complemento de Jenkins para administrar las credenciales de Azure. | 
| [Complemento de Azure Functions](https://plugins.jenkins.io/azure-function)           | Para usar este complemento para implementar en función de Azure, primero debe tener una entidad de servicio de Azure en la instancia de Jenkins. | 
| [Complemento de Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Un Jenkins Plugin para los proyectos de Azure Service Fabric con Linux. |
| [Complemento de Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | Un complemento para cargar artefactos de compilación en o descargar crear dependencias de almacenamiento de blobs de Microsoft Azure. | 
| [Complemento de agentes de máquina virtual de Azure](https://plugins.jenkins.io/azure-vm-agents)         | Un Jenkins Plugin para crear a agentes de Jenkins en Azure virtual machines (a través de la plantilla de Azure Resource Manager). | 
| [Complemento del conjunto de escalado de máquina virtual de Azure](https://plugins.jenkins.io/azure-vmss)           | Establece un complemento de Jenkins para implementar imágenes de máquina virtual en la escala de máquina virtual de Azure. |
