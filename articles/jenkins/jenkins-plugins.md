# <a name="jenkins-plugins-for-azure"></a>Complementos de Jenkins para Azure

Los siguientes complementos de Jenkins admiten varias características para su uso con Azure.
  
| Complemento de Jenkins | DESCRIPCIÓN                                   |
|------------------------------------------------------------------------------|
| [Complemento de Azure App Service](https://plugins.jenkins.io/azure-app-service)     | Complemento de Jenkins para implementar una instancia de Azure App Service (actualmente solo admite la aplicación web). | 
| [Complemento de Azure AD](https://plugins.jenkins.io/azure-ad)                       | Complemento de Jenkins que admite la autenticación y autorización a través de Azure Active Directory. | 
| [Complemento Azure Artifact Manager](https://plugins.jenkins.io/azure-artifact-manager) | El complemento Azure Artifact Manager es un administrador de artefactos que le permite almacenar artefactos en Azure Blob Storage. El complemento Azure Artifact Manager funciona de forma transparente con Jenkins y sus trabajos, es como el administrador de artefactos predeterminado. | 
| [Complemento de Azure Container Agents](https://plugins.jenkins.io/azure-container-agents) | El complemento Azure Container Agents le ayuda a ejecutar un contenedor como un agente en Jenkins. | 
| [Complemento de la tarea de Azure Container Registry](https://plugins.jenkins.io/azure-container-registry-tasks)       | El complemento de Jenkins para enviar una solicitud de compilación de docker para [Azure Container Registry](/azure/container-registry/container-registry-tasks-overview). |
| [Complemento de Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Complemento de Jenkins para implementar configuraciones en Azure Container Service (AKS). | 
| [Complemento de Azure Credential](https://plugins.jenkins.io/azure-credentials)      | Complemento de Jenkins para administrar las credenciales de Azure. | 
| [Complemento de Azure Functions](https://plugins.jenkins.io/azure-function)           | Complemento de Jenkins para implementar una función de Azure Functions. | 
| [Complemento de Azure IoT Edge](https://plugins.jenkins.io/azure-iot-edge)           | El complemento de Azure IoT Edge facilita la configuración de una canalización de CI/CD para desarrollar una solución de IoT Edge en Jenkins. | 
| [Complemento de Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Complemento de Jenkins para los proyectos de Azure Service Fabric con Linux. |
| [Complemento de Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | complemento para cargar artefactos de compilación en Microsoft Azure Blob Storage o descargar dependencias de compilación desde ahí. | 
| [Complemento de agentes de máquina virtual de Azure](https://plugins.jenkins.io/azure-vm-agents)         | Complemento de Jenkins para crear agentes de Jenkins en máquinas virtuales de Azure (a través de la plantilla de Azure Resource Manager). | 
| [Complemento del conjunto de escalado de máquinas virtuales de Azure](https://plugins.jenkins.io/azure-vmss)           | Complemento de Jenkins para implementar imágenes de máquina virtual en conjuntos de escalado de máquinas virtuales de Azure. |
