# <a name="jenkins-plugins-for-azure"></a>Complementos de Jenkins para Azure

Los siguientes complementos de Jenkins admiten varias características para su uso con Azure.
  
| Complemento de Jenkins | DESCRIPCIÓN                                   |
|------------------------------------------------------------------------------|
| [Complemento de Azure App Service](https://plugins.jenkins.io/azure-app-service)     | Complemento de Jenkins para implementar un servicio de aplicación de Azure (actualmente admite solo la aplicación Web). | 
| [Complemento de Azure AD](https://plugins.jenkins.io/azure-ad)                       | Complemento de Jenkins para que admita la autenticación y autorización a través de Azure Active Directory. | 
| [Complemento de administrador de artefacto de Azure](https://plugins.jenkins.io/azure-artifact-manager) | Complemento de administrador de artefacto de Azure es un administrador de artefacto que permite que almacenar los artefactos en Azure Blob Storage. Complemento de administrador de artefacto de Azure funciona transparente para Jenkins y los trabajos, es como el Administrador de artefacto predeterminado. | 
| [Complemento de Azure Container Agents](https://plugins.jenkins.io/azure-container-agents) | Complemento de Azure Container Agents puede ayudarle a ejecutar un contenedor como un agente en Jenkins | 
| [Complemento de la tarea de registro de contenedor de Azure](https://plugins.jenkins.io/azure-container-registry-tasks)       | Complemento de Jenkins para enviar una solicitud de compilación de docker para [Azure Container Registry](/azure/container-registry/container-registry-tasks-overview). |
| [Complemento de Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Complemento de Jenkins para implementar configuraciones de Azure Container Service (AKS). | 
| [Complemento de Azure Credential](https://plugins.jenkins.io/azure-credentials)      | Complemento de Jenkins para administrar las credenciales de Azure. | 
| [Complemento de Azure Functions](https://plugins.jenkins.io/azure-function)           | Complemento de Jenkins para implementar una función de Azure. | 
| [Complemento de Azure IoT Edge](https://plugins.jenkins.io/azure-iot-edge)           | Complemento de Azure IoT Edge facilita la configuración de una canalización de CI/CD para desarrollar una solución de IoT Edge en Jenkins. | 
| [Complemento de Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Complemento de Jenkins para los proyectos de Azure Service Fabric con Linux. |
| [Complemento de Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | complemento para cargar artefactos de compilación en o descargar crear dependencias de almacenamiento de blobs de Microsoft Azure. | 
| [Complemento de agentes de máquina virtual de Azure](https://plugins.jenkins.io/azure-vm-agents)         | Complemento de Jenkins para crear a agentes de Jenkins en Azure virtual machines (a través de la plantilla de Azure Resource Manager). | 
| [Complemento del conjunto de escalado de máquina virtual de Azure](https://plugins.jenkins.io/azure-vmss)           | Establece el complemento de Jenkins para implementar imágenes de máquina virtual en la escala de máquina virtual de Azure. |
