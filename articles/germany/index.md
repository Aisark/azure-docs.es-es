# <a name="azure-germany-documentation"></a>Documentación sobre Azure Alemania

<p style="font-size: 1.125rem; padding-bottom: 0px;">Microsoft Azure Alemania se basa en los principios de seguridad, privacidad, cumplimiento y transparencia que rigen la "nube de confianza" de Microsoft. Aporta residencia de datos en tránsito y en reposo en Alemania y replicación de datos entre centros de datos alemanes para la continuidad empresarial.</p>
<p style="font-size: 1.125rem;">Los datos del cliente que están en los dos centros de datos están bajo el control de un administrador de datos, T-Systems International. Este administrador de confianza es una empresa alemana, filial de Deutsche Telekom. Proporciona controles adicionales para los datos de los clientes, ya que el acceso se proporciona solo con el permiso de los clientes o del administrador de los datos.</p>
<p style="font-size: 1.125rem; padding-bottom: 36px;">Los servicios comerciales en la nube de Microsoft de estos nuevos centros de datos se adhieren a las normas alemanas relativas al control de datos y ofrecen a los clientes opciones adicionales acerca de cómo y dónde se procesan los datos.</p>
