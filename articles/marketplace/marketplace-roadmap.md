# <a name="azure-marketplace-and-appsource-roadmap"></a>Hoja de ruta de Azure Marketplace y AppSource

En este documento se proporciona información sobre las próximas novedades de Azure Marketplace, AppSource y programas de asociados relacionados. Muestra algunas de las características importantes que nos hemos comprometido, un período de tiempo aproximado para cuando puede esperar verlas y se actualizará mensualmente. No es una lista completa de todas las características nuevas, pero está diseñado para proporcionar visibilidad en nuestras principales inversiones. Estos conjuntos de características y períodos de entrega son actuales y sujeta a cambios.

Agradecemos sus sugerencias. Únase a la conversación de [Microsoft Partner Community](https://www.microsoftpartnercommunity.com/) para obtener las actualizaciones más recientes sobre nuevas funcionalidades, programas y eventos.
