-   El logotipo pequeño (48 x 48 píxeles) aparece en los resultados de búsqueda de Azure Marketplace.
-   El logotipo mediano (90 × 90 píxeles) aparece cuando se crea un nuevo recurso en Microsoft Azure.
-   El logotipo grande (entre 216 x 216 y 350 x 350 píxeles) aparece en la página de catálogo de ofertas de Azure Marketplace.
