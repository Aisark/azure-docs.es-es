# <a name="lcs-subscription"></a>Suscripción a LCS

En este tutorial, aprenderá a crear una suscripción. Esta suscripción le proporcionará un inquilino de prueba de Microsoft Online y un proyecto de Microsoft Dynamics Lifecycle Services en el que podrá implementar un entorno. También configurará usuarios adicionales en su inquilino de Microsoft Online y adquirirá experiencia con las siguientes funcionalidades de administración de servicios. Estas son las aptitudes que aprenderá:

- Suscripción y creación de un nuevo inquilino de prueba de Microsoft Online.
- Navegación a proyectos de Lifecycle Services.
- Uso de características en Lifecycle Services.
- Adición de usuarios adicionales a Microsoft Azure Active Directory y al cliente.
- Visualización de recursos en el correo electrónico de suscripción.

## <a name="key-terms"></a>Términos clave

**Inquilino de Microsoft Online Services**: un inquilino es el grupo de todas las suscripciones y usuarios de la organización. Se crea el inquilino al mismo tiempo que la primera suscripción de Microsoft Online Services.

**Suscripción**: una suscripción le proporciona un entorno en línea. Este entorno le permite experimentar Dynamics 365 para operaciones en la nube y también le permite ver cómo las personalizaciones que desarrolla pueden implementarse en la nube.

**Microsoft Azure Active Directory**: El entorno de la nube incluye Azure Active Directory (AD), que le ayuda a administrar usuarios, grupos, roles de seguridad y licencias para aplicaciones en línea, de forma similar a la forma en que las administra para entornos locales.

**Usuarios**: los usuarios de los servicios a los que su organización se ha suscrito se administran en Azure AD. Cualquier usuario de su inquilino puede agregarse y asignarse a roles de seguridad.

**Desarrolladores y administradores**: los desarrolladores y administradores son usuarios que también tienen acceso a Lifecycle Services que les permiten administrar proyectos y entornos. Estos usuarios también son usuarios finales.

Consulte [Suscripción a LCS](https://docs.microsoft.com/dynamics365/operations/dev-itpro/dev-tools/sign-up-preview-subscription) para más información.
