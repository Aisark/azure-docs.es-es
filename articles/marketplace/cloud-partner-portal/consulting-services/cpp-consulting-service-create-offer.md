# <a name="create-a-consulting-offer"></a>Creación de una oferta de consultoría

En este artículo se describe cómo crear y publicar una oferta de servicios de consultoría.

## <a name="define-your-offer"></a>Definición de la oferta

Definición del paquete de ofertas de servicios de consultoría Céntrese en ofertas de ámbito fijo, duración fija, precio estimado fijo (o gratis) y principalmente orientadas a la preventa para un único cliente. Seleccione paquetes de compromisos reiterativos que son populares y eficaces a la hora de impulsar el negocio.

## <a name="create-a-new-offer"></a>Crear una nueva oferta

Para crear una nueva oferta, siga estos pasos.
1. En el menú principal de Cloud Partner Portal, seleccione **Nueva oferta**.
1. En el menú **Nueva oferta**, seleccione **Servicios de consultoría**.

    ![Creación de una oferta de servicios de consultoría](media/cppselectnewconsultingoffer.png)
