# <a name="cloud-partner-portal-management"></a>Administración de Cloud Partner Portal

En esta sección se describen los tipos de operaciones disponibles para crear y administrar las cuentas necesarias para utilizar [Cloud Partner Portal](https://cloudpartner.azure.com).  Estas cuentas son necesarias para [Azure Marketplace](https://azuremarketplace.microsoft.com) y [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Información de contexto

Antes de usar Cloud Partner Portal, debe familiarizarse con los siguientes artículos:

- [Términos y directivas de Azure Marketplace](../../policies-terms.md) enumera los distintos requisitos legales y técnicos y las reglas que se aplican a los usuarios y los publicadores en [Azure Marketplace](https://azuremarketplace.microsoft.com) y [AppSource](https://appsource.microsoft.com).
- [Convertirse en un publicador de Marketplace en la nube](https://docs.microsoft.com/azure/marketplace/become-publisher) proporciona una visión general de todos los pasos necesarios para convertirse en un publicador oficial en la nube de Microsoft.


## <a name="account-creation-and-management"></a>Creación y administración de cuentas

Esta sección contiene los siguientes artículos que explican cómo administrar cuentas de publicador de Marketplace.

- [Requisitos previos para administrar cuentas](./cpp-manage-prerequisites.md) explica cómo convertirse en publicador en la nube registrado, el requisito para utilizar Cloud Partner Portal.
- [Administración de Azure Marketplace y el perfil de publicador de AppSource](./cpp-manage-publisher-profile.md) explica cómo usar Cloud Partner Portal para ver y administrar el perfil de publicador de Marketplace de Microsoft. 
- [Administración de usuarios en Cloud Partner Portal](./cpp-manage-users.md) explica cómo agregar usuarios a la cuenta de publicación en la nube de la organización y cómo administrar su acceso, para permitir que varias personas trabajen en las mismas ofertas.


## <a name="next-steps"></a>Pasos siguientes

Debe cumplir los [Requisitos previos para administrar cuentas](./cpp-manage-prerequisites.md) antes de continuar.  Después, visite [Paseo por Cloud Partner Portal](../portal-tour/cpp-portal-tour.md) para familiarizarse con la interfaz de usuario del portal.
