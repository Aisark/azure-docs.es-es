# <a name="cloud-partner-portal-management"></a>Administración de cloud Partner Portal

En esta sección se describe los tipos de operaciones disponibles para crear y administrar cuentas necesarias para utilizar el [Cloud Partner Portal](https://cloudpartner.azure.com).  Estas cuentas son necesarias para la [Azure Marketplace](https://azuremarketplace.microsoft.com) y [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Información de contexto

Antes de usar Cloud Partner Portal, debe familiarizarse con los siguientes artículos:

- [Términos y directivas de Marketplace de Azure](../../policies-terms.md) enumera los distintos requisitos legales y técnicos y las reglas que se aplican a los usuarios de y los publicadores para el [Azure Marketplace](https://azuremarketplace.microsoft.com) y [AppSource](https://appsource.microsoft.com).
- [Convertirse en anunciante de Marketplace en la nube](../../become-publisher.md) proporciona una visión general de todos los pasos necesarios para convertirse en un publicador de nube de Microsoft oficial.


## <a name="account-creation-and-management"></a>Creación de cuentas y administración

Esta sección contiene los siguientes artículos que explican cómo administrar cuentas de publicador de marketplace.

- [Requisitos previos para administrar las cuentas](./cpp-manage-prerequisites.md) explica cómo convertirse en publicador registrado en la nube, el requisito para utilizar CLoud Partner Portal.
- [Administrar perfil de anunciante de Azure Marketplace y AppSource](./cpp-manage-publisher-profile.md) explica cómo usar Cloud Partner Portal para ver y administrar su perfil de anunciante de marketplace de Microsoft. 
- [Administración de usuarios en Cloud Partner Portal](./cpp-manage-users.md) agregar usuarios a la nube de su organización cuenta y cómo administrar su acceso, para permitir que varias personas trabajen en las ofertas de la mismas publicación.


## <a name="next-steps"></a>Pasos siguientes

Debe cumplir la [requisitos previos para administrar las cuentas](./cpp-manage-prerequisites.md) antes de continuar.  Después, también debe tener la [Cloud Partner Portal Tour](../portal-tour/cpp-portal-tour.md) familiarizarse con la interfaz de usuario del portal.
