# <a name="prerequisites-for-managing-accounts-on-the-cloud-partner-portal"></a>Requisitos previos para administrar cuentas en Cloud Partner Portal 

Para poder usar el [Cloud Partner Portal](https://cloudpartner.azure.com/) para cualquier función, incluida la administración de cuentas, debe seguir una serie de pasos para registrar a sí mismos y su organización como publicadores de Microsoft en la nube.


## <a name="account-terms-and-guidelines"></a>Instrucciones y términos de la cuenta

Revise cuidadosamente los derechos y responsabilidades en la [contrato de servicios de Microsoft](https://www.microsoft.com/servicesagreement) y [Terms de uso de Microsoft](https://www.microsoft.com/legal/intellectualproperty/copyright) antes de crear una cuenta.  

Elija cuidadosamente el nombre de cuenta.  Representa la organización, por lo que no debe contener ninguna palabra con derechos de autor o contenciosa.  Por ejemplo, Microsoft prohíbe el uso de sus términos con derechos de autor, por ejemplo, "Azure" y "Microsoft", los nombres de cuenta.  Para obtener más información, consulte [directrices para Azure Marketplace](../../guidelines.md)


## <a name="registration-steps"></a>Pasos de registro

En la tabla siguiente se enumera los pasos necesarios para registrar completamente como un publicador de nube de Microsoft.  Para obtener instrucciones completas, consulte [convertirse en anunciante de Marketplace en la nube](../../become-publisher.md). 


|                   Paso                   |  Registro resultante o recurso                     |
|                  ------                  |  -----------------------------------                    |
| 1. Registrarse en Microsoft Partner Network |  Un Id. de Microsoft Partner Network para su organización |
| 2. Crear una cuenta de Microsoft *           |  Un comprobado la cuenta de Microsoft, terminan en @outlook.com, @live.com, etcetera. |
| 3. Envío del formulario de registro de Marketplace | Una cuenta de publicador en Cloud Partner Portal      |
| 4. Conexión a Cloud Partner Portal        | Autoridad para crear y publicar ofertas, administrar usuarios, marketplace recopilar métricas, etcetera. |
| 5. Regístrese en Microsoft Developer Center * | Una cuenta comprobada de centro de desarrollo; autoridad para llevar a cabo las transacciones de marketplace  |
|   |   |

\* Estos pasos son necesarios para Azure Marketplace transact ofertas, pero se recomienda para todas las ofertas.  Una vez que se establezca una cuenta de Microsoft y registrar como un Microsoft Developer, puede tener acceso a recursos adicionales de cliente y desarrollador de Microsoft, así como ofertas de asociado opcional.  


## <a name="next-steps"></a>Pasos siguientes

Una vez completados los pasos de registro, debe iniciar sesión en el [Cloud Partner Portal](https://cloudpartner.azure.com/).  Puede usar este portal y administrar perfiles de publicador y administrar cuentas de usuario.
