# <a name="audit-history-page"></a>Página Historial de auditoría

La página [**Historial de auditoría**](https://cloudpartner.azure.com/#history) muestra una lista de los eventos de publicación para todas las ofertas de Marketplace.  Las columnas incluyen el tipo de evento, la fecha y hora de aparición, el nombre de la oferta, el publicador y el usuario que inició la acción.  Los resultados se pueden ordenar por columna o filtrarse en una cadena especificada.  Esta página también permite al usuario descargar un archivo de valores separados por comas (.csv) de la lista actual de auditoría.
