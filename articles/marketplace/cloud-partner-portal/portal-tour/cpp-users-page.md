# <a name="users-page"></a>Página Usuarios

La página [**Usuarios**](https://cloudpartner.azure.com/#users) muestra una lista de todos los usuarios registrados de Cloud Partner Portal.  Las columnas incluyen la dirección de correo electrónico de cada usuario, el rol en el portal, el tipo de oferta y la organización de publicación. Los resultados se pueden ordenar por columna o filtrarse en una cadena especificada.  

En esta página también se muestra un botón **Agregar usuario** para agregar usuarios a la organización de publicación.  Para obtener más información, consulte [Administración de usuarios en Cloud Partner Portal](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-users.md).
