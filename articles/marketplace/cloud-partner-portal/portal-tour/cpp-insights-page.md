# <a name="insights-page"></a>Página Insights

La página [**Insights**](https://cloudpartner.azure.com/#insights) proporciona los siguientes conjuntos de paneles para ayudarle a comprender y maximizar las ventas de sus ofertas de Marketplace.  Se accede a cada panel a través de una pestaña del mismo nombre en la parte superior de la página.


|  **Panel**    |  **Contenido mostrado**               |
|  -------------    |  ---------------------               |
| **Resumen**       | gráficos, tendencias y valores de datos agregados que resumen la actividad de Marketplace para las ofertas del publicador |
| **Pago**        | pagos y transacciones relacionadas en formatos gráficos y descargables |
| **Pedidos y uso** | información de uso y compras en formatos gráficos y descargables |
| **Cliente**      | información del cliente, incluido su perfil de compra |
| **Implementación**    | información de implementación correcta y con errores en formatos gráficos y de nivel de evento |
| **Descargas**     | lista de solicitudes de descarga de los últimos 30 días |
| **Analytics**     | resumen de análisis web y análisis de rendimiento de campañas |
|  |  |


## <a name="summary-dashboard"></a>Panel Resumen
El panel Resumen es la primera pestaña y es la pestaña predeterminada al navegar a la página Insights.  Este diseño intencionado refleja que este panel proporciona a los publicadores la vista más amplia de la actividad de ventas de sus ofertas.  Proporciona representaciones gráficas de las tendencias de uso normalizado; el uso mensual; las tendencias de uso por mercado geográfico, oferta y cliente; y la tendencia de pagos.  La salida se puede personalizar mediante la especificación de fechas de inicio y final para el análisis.
