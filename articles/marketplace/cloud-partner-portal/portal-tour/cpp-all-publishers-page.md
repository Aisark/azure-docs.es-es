# <a name="all-publishers-page"></a>Página Todos los publicadores

En la página [**Todos los publicadores**](https://cloudpartner.azure.com/#publishers) se muestra una lista de todos los publicadores de nube de Microsoft registrados, incluido su nombre único de publicador y el nombre para mostrar. Puede ordenar los resultados por columna o especificar una cadena para filtrar los resultados.  En esta página también se muestra un botón **Agregar publicador**, pero este botón solo está habilitada para los administradores del portal.  Si selecciona un publicador de esta lista, se mostrará el perfil para ese publicador.  Para obtener más información, consulte [Administración del perfil de anunciante](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-publisher-profile.md).
