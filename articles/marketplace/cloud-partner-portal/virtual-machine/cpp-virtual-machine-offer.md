# <a name="azure-virtual-machine-offer"></a>Oferta de máquina virtual de Azure

[Cloud Partner Portal](https://cloudpartner.azure.com/) ya no admite la creación ni la administración de ofertas de máquinas virtuales de Azure. Esta funcionalidad se ha movido al [Centro de partners](https://partner.microsoft.com/pcv/) de Microsoft. Para obtener más información, consulte [Creación de una oferta de máquina virtual de Azure](../../partner-center-portal/azure-vm-create-offer.md).
