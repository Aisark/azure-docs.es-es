# <a name="container-publishing-prerequisites"></a>Requisitos previos de publicación de contenedores

En este artículo se describen los requisitos previos para la publicación de una oferta de contenedor en Azure Marketplace.  Si aún no lo ha hecho, debe revisar la [Guía de publicación de ofertas de contenedores](../../marketplace-containers.md).

## <a name="publishing-prerequisites"></a>Requisitos previos de publicación

Para publicar una nueva imagen de contenedor, se deben cumplir los siguientes requisitos previos:

- Acceda a Cloud Partner Portal. Para más información, consulte la guía de publicación de Azure Marketplace y AppSource.
- Aceptación de los Términos de Azure Marketplace
- Hospede el recurso técnico de contenedor en una instancia de Azure Container Registry.
- Tenga los metadatos del contenedor listos para su uso. Por ejemplo, algunos de estos:
  - Un título
  - Una descripción (en formato HTML)
  - Una imagen de logotipo (en formato PNG) y en estos tamaños de imagen fijos: 40x40 px, 90x90 px, 115x 115 px y 255x115 px.
- Informes de *Términos de uso* y *directiva de privacidad*
- Documentación de soluciones de Container
- Contactos de soporte técnico

## <a name="next-steps"></a>Pasos siguientes

Si aún no lo ha hecho, debe [preparar los recursos técnicos del contenedor](./cpp-create-technical-assets.md) antes de [crear su oferta de contenedor](./cpp-create-offer.md).
