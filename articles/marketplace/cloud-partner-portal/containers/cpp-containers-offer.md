# <a name="containers"></a>Contenedores

[Cloud Partner Portal](https://cloudpartner.azure.com/) ya no admite la creación y la administración de ofertas de contenedor de Azure. Esta funcionalidad se ha movido al [Centro de partners](https://partner.microsoft.com/pcv/) de Microsoft. Para más información, consulte [Creación de una oferta de contenedor de Azure](../../partner-center-portal/create-azure-container-offer.md).
