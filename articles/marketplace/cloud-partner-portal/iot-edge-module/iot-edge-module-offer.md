# <a name="iot-edge-module-offer"></a>Oferta de módulo de IoT Edge

[Cloud Partner Portal](https://cloudpartner.azure.com/) ya no admite la creación y administración de ofertas de módulo de IoT Edge. Esta funcionalidad se ha movido al [Centro de partners](https://partner.microsoft.com/pcv/) de Microsoft. Para obtener más información, consulte [Creación, configuración y publicación de una oferta de módulo de IoT Edge en Azure Marketplace](../../partner-center-portal/azure-iot-edge-module-creation.md).
