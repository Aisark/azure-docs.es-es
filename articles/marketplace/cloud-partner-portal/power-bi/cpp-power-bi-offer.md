# <a name="power-bi-app-offer"></a>Oferta de aplicación de Power BI

Cloud Partner Portal ya no admite la creación y administración de ofertas de aplicación de Power BI. Esta funcionalidad se ha movido al [Centro de partners](https://partner.microsoft.com/pcv/) de Microsoft. Para obtener más información, consulte [Creación de una aplicación de Power BI para Microsoft AppSource](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer).
