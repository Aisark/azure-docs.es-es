# <a name="create-power-bi-app-offer-technical-assets"></a>Creación de los recursos técnicos de la oferta de aplicación de Power BI

>[!Important]
>A partir del 13 de abril de 2020, se comenzará a trasladar la administración de las ofertas de aplicaciones de Power BI al Centro de partners. Después de la migración, las ofertas se crearán y administrarán en el Centro de partners. Siga las instrucciones que se indican en [Información general sobre la creación de aplicaciones de Power BI](https://aka.ms/AzureCreatePBIServiceApp) para administrar las ofertas migradas.

El recurso técnico principal que necesitará para este tipo de oferta es una aplicación de Power BI. Una aplicación de Power BI es una colección de conjuntos de datos principales, informes o paneles, servicios conectados opcionales y conjuntos de datos insertados. Ha evolucionado a partir de un tipo de recurso conocido anteriormente como [paquete de contenido](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction). Para más información sobre el desarrollo de este tipo de aplicación, consulte [¿Cuáles son las aplicaciones de la plantilla de Power BI?](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtain-an-installation-url"></a>Obtención de una dirección URL de instalación

Solo puede compilar una aplicación de Power BI dentro del [servicio Power BI](https://powerbi.microsoft.com/). Para comenzar, inicie sesión con una [licencia de Power BI Pro](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Una vez creada y probada la aplicación de servicio en Power BI, guarde la dirección URL de instalación de la aplicación que generó Power BI. Por último, debe proporcionar esta dirección URL en la pestaña [Información técnica](./cpp-technical-info-tab.md) del formulario de envío de Cloud Partner Portal.
 

## <a name="next-steps"></a>Pasos siguientes

Después de crear y probar la aplicación de Power BI y de crear la oferta asociada, puede [publicar la oferta de la aplicación de Power BI](./cpp-publish-offer.md).
