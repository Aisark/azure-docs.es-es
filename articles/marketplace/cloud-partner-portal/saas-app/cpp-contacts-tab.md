# <a name="saas-application-contacts-tab"></a>Pestaña Contactos de una aplicación SaaS

En este artículo se describe cómo usar la pestaña Contactos para identificar los contactos para los clientes que usen su oferta.


## <a name="identify-contacts"></a>Identificación de los contactos

Use la siguiente captura de pantalla como guía para proporcionar la información necesaria. Todos los campos marcados con un asterisco (*) debe completarse.

![Pestaña Contactos](./media/saas-offer-contacts.png)

1. En Contacto de ingeniería, proporcione información para los campos siguientes:

   - **Nombre\***  -nombre del contacto de ingeniería para la aplicación. Este contacto recibirá comunicaciones técnicas de Microsoft.
   - **Correo electrónico\***  -dirección del contacto de ingeniería de correo electrónico.
   - **Teléfono\***  -número de teléfono de contacto de ingeniería. Se admiten notaciones de número de teléfono ISO. Para más información, consulte [E.123](https://en.wikipedia.org/wiki/E.123).
  
2. En Contacto de soporte técnico, proporcione información para los campos siguientes:

   - **Nombre\***  -nombre del contacto de soporte técnico para la aplicación. Este contacto recibirá comunicaciones relacionadas con el soporte técnico de Microsoft.
   - **Correo electrónico\***  -dirección del contacto de soporte técnico para la aplicación de correo electrónico.
   - **Teléfono\***  -número de teléfono de contacto de soporte técnico. Se admiten notaciones de número de teléfono ISO. Para más información, consulte [E.123](https://en.wikipedia.org/wiki/E.123).
   - **Dirección URL de soporte\***  -dirección URL a la página de soporte técnico.

3. Seleccione **Guardar** cuando termine con los formularios.
