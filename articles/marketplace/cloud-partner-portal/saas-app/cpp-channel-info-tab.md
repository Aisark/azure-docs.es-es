# <a name="saas-application-channel-info-tab"></a>Pestaña Channel Info de una aplicación SaaS

Use la pestaña Channel Info (Información de canal) para proporcionar información de canal para su aplicación, incluidos los materiales de lanzamiento al mercado (GTM) y la información de contacto.

![Formulario Channel Info](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>Para configurar la información del canal

1. En **Channel Info>GTM Materials** (Materiales de lanzamiento al mercado), escriba la dirección URL para el sitio web que hospeda los materiales de lanzamiento al mercado.
2. En **Channel Manager Contact** (Contacto con el administrador de canal), proporcione un contacto de administrador con el que Microsoft pueda comunicarse por soporte técnico y asuntos empresariales. Escriba la información de estos campos: Nombre, Dirección de correo electrónico y Número de teléfono.
3. Seleccione **Guardar** para finalizar la configuración de la información del canal.
