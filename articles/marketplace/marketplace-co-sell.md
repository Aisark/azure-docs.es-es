# <a name="co-sell-partner-engagement"></a>Participación de asociados de venta conjunta

Si es un asociado, puede usar el programa de venta conjunta de Microsoft para trabajar directamente con Microsoft en oportunidades de venta conjunta. El programa ayuda a nuestros asociados a llegar a una gran comunidad de clientes y crear nuevas oportunidades de ventas que pueden ayudar a acelerar el crecimiento del negocio.

## <a name="partner-selling-motions"></a>Movimientos de ventas de asociados

Microsoft ofrece tres movimientos de ventas de asociados:

* *Venta mediante un asociado*: puede revender la tecnología de Microsoft a través de sus propios vendedores.
* *Venta conjunta de servicios*: los vendedores y asociados colaboran para vender su proyecto o servicios administrados con tecnología de Microsoft incrustada.
* *Venta conjunta de IP*: los vendedores o asociados colaboran para vender su solución de IP empaquetada y repetible*.

\* Algunas de estas soluciones de venta conjunta de IP pueden recibir incentivos adicionales al venderse e implementarse correctamente con los equipos de ventas de campo de Microsoft.
