# <a name="consulting-service-prerequisites"></a>Requisitos previos de los servicios de consultoría

Para obtener información sobre la oferta de servicios de consultoría, incluidos los requisitos de este tipo de oferta, consulte [Servicios de consultoría de Microsoft AppSource y Azure Marketplace](../consulting-services.md). En la siguiente tabla se muestran los requisitos adicionales para tipos específicos de servicios de consultoría.

| Tipo de servicio de consultoría | Producto principal que se selecciona en el Centro de partners | Requisito |
|---------|---------|---------|
| Productos de Azure&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;| Azure | La oferta debe enumerar al menos una de las siguientes competencias obtenidas completamente: <ul><li>Desarrollo de aplicaciones</li><li>Integración de aplicaciones</li><li>Administración del ciclo de vida de las aplicaciones</li><li>Plataforma de nube</li><li>Análisis de datos</li><li>Centro de datos</li><li>Plataforma de datos</li><li>DevOps|
| Dynamics 365 Sales<br>Dynamics 365 Marketing<br>Dynamics 365 Customer Service<br>Dynamics 365 Field Service<br>Dynamics 365 Human Resources | Aplicaciones de Customer Engagement | Debe tener la certificación Gold o Silver en la [competencia Business Applications en la nube](https://partner.microsoft.com/membership/cloud-business-applications-competency) para la opción Customer Engagement, o tener una oferta de venta conjunta para el producto principal con el que está relacionado el servicio. |
| Dynamics 365 Finance<br>Dynamics 365 Supply Chain Management<br>Dynamics 365 Commerce<br>Dynamics 365 Human Resources<br>Dynamics 365 Project Service Automation | Aplicaciones de Finance and Operations | Debe tener la certificación Gold o Silver en la [competencia Business Applications en la nube](https://partner.microsoft.com/membership/cloud-business-applications-competency) para la opción Unified Operations, o tener una oferta de venta conjunta para el producto principal con el que está relacionado el servicio. |
| Dynamics 365 Customer Insights | Dynamics 365 Customer Insights | Debe tener al menos una implementación en producción correcta de [Dynamics 365 Customer Insights](https://dynamics.microsoft.com/ai/customer-insights/) con al menos cinco medidas y cinco segmentos, o tener una oferta de venta conjunta para el producto principal con el que está relacionado el servicio. |
| Dynamics 365 Business Central | Dynamics 365 Business Central | Debe tener una certificación Gold o Silver en la [competencia Planificación de recursos empresariales](https://partner.microsoft.com/membership/enterprise-resource-planning-competency) y prestar servicio al menos a tres clientes o haber publicado una aplicación Business Central en Microsoft AppSource o tener una oferta de venta conjunta para el producto principal con el que está relacionado el servicio. |
| Power BI | Power BI | Debe aparecer en la [presentación del partner de Power BI](https://powerbi.microsoft.com/partner-showcase/), o tener una oferta de venta conjunta para el producto principal con el que está relacionado el servicio.|
|PowerApps | PowerApps | Debe ser elegible para aparecer en el apartado de ventajas avanzadas del programa [Power Apps Partnership](https://aka.ms/PowerAppsPartner), o tener una oferta de venta conjunta para el producto principal con el que está relacionado el servicio. |

Cuando esté listo para crear la oferta, consulte [Creación de una oferta de servicios de consultoría](./create-consulting-service-offer.md).
