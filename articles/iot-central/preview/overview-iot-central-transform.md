# <a name="transform-your-iot-data-preview-features"></a>Transformación de los datos de IoT (Características en vista previa (GB))

[!INCLUDE [iot-central-pnp-original](../../../includes/iot-central-pnp-original-note.md)]

Como plataforma de aplicaciones, IoT Central proporciona varias facetas clave para ayudarle a transformar los datos de IoT en conclusiones empresariales que impulsan resultados útiles. IoT Central proporciona métodos para ampliar los datos de IoT a sistemas externos con el fin de crear integraciones con aplicaciones de línea de negocio y aplicaciones personalizadas. Puede supervisar el estado de los dispositivos y las operaciones mediante la creación de reglas que notifiquen a los técnicos a través de un servicio móvil. Puede generar conclusiones empresariales específicas con el análisis personalizado y el aprendizaje automático mediante la exportación de datos de telemetría sin procesar a los servicios en Azure. Puede crear servicios y herramientas que supervisen y controlen dispositivos y administren su aplicación de IoT Central con las API públicas. 

![Información general sobre la transformación en IoT Central](media/overview-iot-central-transform/transform.png)

## <a name="monitor-device-health-and-operations-using-rules"></a>Supervisión del estado de los dispositivos y las operaciones mediante reglas
Una vez que las máquinas estén conectadas y enviando datos, puede identificar qué máquinas tienen problemas o envían mensajes de error para que se puedan corregir de modo que se ejecuten normalmente con un tiempo de inactividad mínimo. Puede crear reglas en la aplicación IoT Central para supervisar la telemetría que los dispositivos envían y avisarle cuando se supere un umbral o se envíe un mensaje de evento específico. Puede configurar acciones (por ejemplo, acciones de correo electrónico y webhooks) para las reglas a fin de notificar a las personas adecuadas y los sistemas de nivel inferior correctos. Obtenga más información sobre las reglas aquí.

## <a name="run-custom-analytics-and-processing-on-your-exported-data"></a>Ejecución de análisis y procesamiento personalizados en los datos exportados
Puede generar conclusiones empresariales muy personalizadas, como determinar las tendencias de eficiencia de la máquina o predecir el uso de energía futuro en una fábrica mediante la creación de canalizaciones de análisis personalizadas para procesar la telemetría de IoT sin procesar y almacenar el resultado final. Puede crear exportaciones de datos en la aplicación IoT Central para exportar datos de telemetría, propiedades y ciclo de vida del dispositivo, y cambios en la plantilla de dispositivos a otros servicios de Azure. Así, puede analizar, almacenar y visualizar los datos de las herramientas que más le interesen. Obtenga más información sobre la exportación de datos aquí.

## <a name="build-custom-iot-solutions-and-integrations-with-apis"></a>Creación de soluciones de IoT personalizadas e integraciones con las API
Puede compilar soluciones de IoT (por ejemplo, aplicaciones tipo Complemento de teléfono) que pueden controlar dispositivos de forma remota y configurar nuevos dispositivos o integraciones personalizadas con aplicaciones de línea de negocio existentes para interactuar con los dispositivos y datos de IoT que se adaptan a su negocio. Puede usar IoT Central como base para el modelado de dispositivos, la incorporación, la administración y el acceso a datos. Obtenga más información sobre la API pública en este módulo de aprendizaje.
