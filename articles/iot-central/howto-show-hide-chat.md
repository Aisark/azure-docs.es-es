# <a name="toggle-live-chat"></a>Chat en directo de alternancia

Este artículo de procedimientos que muestra cómo alternar el chat en directo en la aplicación IoT Central. Puede utilizar el chat en directo para tener acceso a soporte técnico.

> [!NOTE]
> La opción de chat está disponible solo para las suscripciones de prueba.

## <a name="chat-with-us"></a>Chatee con nosotros

Para obtener soporte técnico, abra el IoT Central, aplicación y seleccione **chatee con nosotros**.

![Chatee con nosotros](media/howto-show-hide-chat/chat-with-us.png)

Puede escribir una pregunta, como se muestra en la captura de pantalla siguiente:

![El formulario "Configure Device Details" (Configurar detalles del dispositivo) con detalles de la configuración y las propiedades](media/howto-show-hide-chat/sample-chat.png)

## <a name="hide-chat"></a>Ocultar chat

Para ocultar el chat, elija **ocultar Chat** en el **ayuda** panel:

 ![Ocultar chat](media/howto-show-hide-chat/hide-chat.png)

## <a name="enable-chat"></a>Habilitar la charla

Para mostrar el chat, elija **Mostrar Chat** en el **ayuda** panel:

 ![Mostrar chat](media/howto-show-hide-chat/show-chat.png)
