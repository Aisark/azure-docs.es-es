# <a name="about-azure-edge-zones---preview"></a>Acerca de Azure Edge Zones (versión preliminar)

Azure Edge Zones es una familia de ofertas de Microsoft Azure que permite el procesamiento de datos cerca del usuario. Puede implementar máquinas virtuales, contenedores y otros servicios de Azure seleccionados en Azure Edge Zone para satisfacer los requisitos de baja latencia y alto rendimiento de las aplicaciones.

Los escenarios de casos de uso típicos de las zonas perimetrales incluyen:

- Comando y control en tiempo real en robótica
- Análisis e inferencia en tiempo real con inteligencia artificial y aprendizaje automático
- Visión por ordenador
- Representación remota para escenarios de realidad mixta y VDI
- Juegos de inmersión de varios jugadores
- Streaming multimedia y entrega de contenido
- Vigilancia y seguridad

Azure Edge Zones dispone de tres ofertas independientes:

- Azure Edge Zones
- Azure Edge Zones with Carrier
- Azure Private Edge Zones

## <a name="azure-edge-zones"></a><a name="edge-zones"></a>Azure Edge Zones

![Edge Zones](./media/edge-zones-overview/edge-zones.png "Edge Zones")

Azure Edge Zones son pequeñas extensiones de superficie de Azure ubicadas en centros de población que están lejos de las regiones de Azure. Azure Edge Zone admite máquinas virtuales, contenedores y un conjunto seleccionado de servicios de Azure que permiten ejecutar aplicaciones sensibles a la latencia y el rendimiento cerca de los usuarios finales. Azure Edge Zones forma parte de la red global de Microsoft y ofrece una conectividad segura, confiable y de alto ancho de banda entre las aplicaciones que se ejecutan en la zona perimetral cerca del usuario y el conjunto completo de servicios de Azure que se ejecutan en las regiones de Azure. Azure Edge Zones es propiedad de Microsoft, quien lleva su operativa y permite usar el mismo conjunto de herramientas y portal de Azure para administrar e implementar servicios en las zonas perimetrales.

Entre los casos de uso comunes se incluyen:

- Juegos y streaming de juegos
- Streaming multimedia y entrega de contenido
- Análisis e inferencia en tiempo real con inteligencia artificial y aprendizaje automático
- Representación para realidad mixta

Azure Edge Zones estará disponible en las siguientes áreas metropolitanas:

- Nueva York, NY
- Los Angeles, CA
- Miami, FL

[Póngase en contacto con el equipo de Edge Zones](https://aka.ms/EdgeZones) para más información.

## <a name="azure-edge-zones-with-carrier"></a><a name="carrier"></a>Azure Edge Zones with Carrier

![Edge Zones with Carrier](./media/edge-zones-overview/edge-carrier.png "Edge Zones with Carrier")

Azure Edge Zones with Carrier son pequeñas extensiones de superficie de Azure ubicadas en centros de datos de operadores móviles de centros de población. La infraestructura de Azure Edge Zones with Carrier se sitúa a un solo salto de la red 5G del operador móvil, lo que ofrece una latencia de menos de 10 milisegundos a las aplicaciones de dispositivos móviles. Azure Edge Zones with Carrier se implementa en los centros de datos de los operadores móviles y se conecta a la red global de Microsoft. Ofrece una conectividad segura, confiable y de alto ancho de banda entre las aplicaciones que se ejecutan cerca del usuario y el conjunto completo de servicios de Azure que se ejecutan en las regiones de Azure. Los desarrolladores pueden usar el mismo conjunto de herramientas conocidas para crear e implementar servicios en Edge Zones.

Entre los casos de uso comunes se incluyen:

- Juegos y streaming de juegos
- Streaming multimedia y entrega de contenido
- Análisis e inferencia en tiempo real con inteligencia artificial y aprendizaje automático
- Representación para realidad mixta
- Automóviles conectados
- Telemedicina

Edge Zones se ofrecerá en colaboración con los operadores siguientes:

- AT&T (Atlanta, Dallas y Los Ángeles)

## <a name="azure-private-edge-zones"></a><a name="private-edge-zones"></a>Azure Private Edge Zones

![Private Edge Zones](./media/edge-zones-overview/private-edge.png "Private Edge Zones")

Azure Private Edge Zones son pequeñas extensiones de superficie de Azure ubicadas en el entorno local. Azure Private Edge Zones se basa en la plataforma de [Azure Stack Edge](https://azure.microsoft.com/products/azure-stack/edge/) y permite el acceso de baja latencia a los servicios de proceso y almacenamiento implementados de forma local. Private Edge Zones también le permite implementar funciones de red virtualizadas (VNF), como núcleos de paquetes móviles, enrutadores, firewalls, dispositivos SD-WAN y aplicaciones de proveedores de software independientes, como las [aplicaciones administradas de Azure](https://azure.microsoft.com/services/managed-applications/), en paralelo junto con máquinas virtuales y contenedores locales. Azure Private Edge Zones incluye una solución de orquestación nativa en la nube que le permite administrar el ciclo de vida de las funciones de red virtualizadas (VNF) y las aplicaciones desde Azure Portal.

Azure Private Edge Zones le permite desarrollar e implementar aplicaciones de forma local con las mismas herramientas conocidas utilizadas para crear e implementar aplicaciones en Azure. También puede ejecutar redes móviles privadas (LTE privado, 5G privado), funciones de seguridad como firewalls y ampliar las redes locales entre diversas instalaciones y Azure mediante dispositivos SD-WAN en los mismos dispositivos de Private Edge Zone y administrarlas desde Azure.

Entre los casos de uso comunes se incluyen:

- Comando y control en tiempo real en robótica
- Análisis e inferencia en tiempo real con inteligencia artificial y aprendizaje automático
- Visión por ordenador
- Representación remota para escenarios de realidad mixta y VDI
- Vigilancia y seguridad

Disponemos de un amplio ecosistema de proveedores de VNF, ISV y asociados de MSP para habilitar soluciones de un extremo a otro mediante Private Edge Zones. [Póngase en contacto con el equipo de Private Edge Zones](https://aka.ms/EdgeZonesPartner) para más información.

## <a name="private-edge-zones---solutions"></a><a name="solutions-private-edge"></a>Private Edge Zones: soluciones

### <a name="private-mobile-network-on-private-edge-zones"></a><a name="private-mobile-private-edge"></a>Private Mobile Network en Private Edge Zones

![Private Mobile Network en Private Edge Zones](./media/edge-zones-overview/mobile-networks.png "Private Mobile Network en Private Edge Zones")

Ahora puede implementar una red móvil privada en Private Edge Zones. Las redes móviles privadas permiten una latencia muy baja, gran capacidad y una red inalámbrica confiable y segura que es necesaria para las aplicaciones críticas de la empresa. Las redes móviles privadas pueden habilitar escenarios como el comando y el control de vehículos guiados automatizados (AGV) en un almacén, la comunicación en tiempo real entre robots en una factoría inteligente, realidad aumentada y aplicaciones perimetrales de realidad virtual.

La función de red Virtualized Evolved Packet Core (vEPC) forma el cerebro de una red móvil privada. Ahora puede implementar un vEPC en Private Edge Zones. Para obtener una lista de los asociados de vEPC que están disponibles en Private Edge Zones, consulte [ISV de vEPC](#vEPC).

La implementación de una solución de red móvil privada en Private Edge Zones requiere otros componentes; por ejemplo, puntos de acceso móvil, tarjetas SIM y otras VNF, como los enrutadores. El acceso a un espectro con licencia o sin licencia es fundamental para configurar una red móvil privada. Además, puede que necesite ayuda con la planeación de RF, el diseño físico, la instalación y el soporte técnico. Para obtener una lista de asociados, consulte [Asociados de radio móvil](#mobile-radio).

Microsoft proporciona un ecosistema de asociados que puede ayudar en todos los aspectos de este proceso: desde el planeamiento de la red, la adquisición de los dispositivos necesarios o la configuración del hardware hasta la administración de la configuración desde Azure. Con un conjunto de asociados validados que están estrechamente integrados con Microsoft, puede estar seguro de que la solución será confiable y fácil de usar. Puede centrarse en sus escenarios principales, a la vez que confía en Microsoft y sus asociados para ayudar con el resto.

### <a name="sd-wan-on-private-edge-zones"></a><a name="sdwan-private-edge"></a>SD-WAN en Private Edge Zones

![SD-WAN en Private Edge Zones](./media/edge-zones-overview/sd-wan.png "SD-WAN en Private Edge Zones")
 
SD-WAN como tecnología le permite crear redes de área extensa (WAN) de nivel empresarial con un mayor ancho de banda, acceso de alto rendimiento a la nube, inserción del servicio, confiabilidad, administración de directivas y una amplia visibilidad de la red. SD-WAN proporciona conectividad sin problemas para las sucursales, organizada desde controladores centrales redundantes, con un costo menor de propiedad.
SD-WAN en Private Edge Zones permite apartarse del modelo centrado en gastos de capital hacia un modelo de software como servicio (SaaS) para reducir los presupuestos de TI. Puede usar la opción del orquestador o el controlador de asociados de SD-WAN para habilitar nuevos servicios y propagarlos a toda la red de forma inmediata.
