# <a name="azure-maps-mobility-coverage"></a>Cobertura de mapas movilidad de Azure

Azure Maps [el servicio de movilidad](https://aka.ms/AzureMapsMobilityService) permite la funcionalidad de tránsito pública, uso compartido de bicicletas y recurso compartido de moto enrutamiento con multimodal opciones de planeamiento de carreras. Tránsito pública, ofrece el servicio de búsqueda para estaciones cercanas, se detiene y las líneas y proporciona información como public en tiempo real y programado llegadas, lista de las guías de ruta y se detiene, así como itinerarios de paso a paso detallados y multimodales en tránsito y alertas del servicio. Mobility service también devuelve en tiempo real e información estática de disponibilidad para automóvil compartir los vehículos y compartidos bicicletas y ciclomotores.

Azure Maps no tiene el mismo nivel de información y precisión en todas las ciudades y países o regiones. La posibilidad de llamar a los datos de tránsito depende de la ciudad y los datos no pueden incluir todas las opciones de tránsito pública y agencias que sirven a la ciudad.

En la tabla siguiente proporciona información de cobertura para el servicio de movilidad de Azure Maps.


| Símbolo | Significado |
|--------|---------|
| *      |Cobertura casi completa para el país o región.|
