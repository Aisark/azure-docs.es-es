# <a name="build-real-time-applications-with-azure-signalr"></a>Creación de aplicaciones en tiempo real con Azure SignalR

[Azure SignalR](https://azure.microsoft.com/services/signalr-service/) le permite agregar fácilmente funcionalidad en tiempo real a la aplicación. Esta funcionalidad en tiempo real permite que el servicio inserte las actualizaciones de contenido a los clientes conectados. Como resultado, las aplicaciones se actualizan sin necesidad de sondear el servidor ni de enviar nuevas solicitudes HTTP para las actualizaciones.

## <a name="azure-signalr-features"></a>Características de Azure SignalR
- **Amplia compatibilidad con clientes**: funciona con una amplia gama de clientes tales como exploradores móviles y web, aplicaciones de escritorio, proceso de servidor, dispositivos IoT y las consolas de juegos.
- SignalR Service permite que **varias instancias funcionen conjuntamente** para escalar a millones de conexiones de aplicaciones móviles. También admite varias regiones globales con fines de particionamiento, alta disponibilidad o recuperación ante desastres.
- **Se ofrecen API enriquecidas con distintos patrones de mensajería** para enviar mensajes a una conexión en particular, a todas las conexiones o a un subconjunto de conexiones que pertenecen a un usuario específico o que se han colocado en un grupo arbitrario.
