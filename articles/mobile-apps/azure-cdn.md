# <a name="content-delivery-network-on-azure-with-azure-cdn-service"></a>Content Delivery Network en Azure con el servicio Azure CDN
[Azure CDN](https://azure.microsoft.com/services/cdn/) ofrece a los desarrolladores una solución global para la entrega rápida de contenido de alto ancho de banda (como vídeos, imágenes o archivos estáticos) a los usuarios mediante el almacenamiento en caché de su contenido en cientos de puntos de presencia (PoP) de todo el mundo. Azure CDN también puede acelerar el contenido dinámico que no puede almacenarse en caché y le permite reducir los tiempos de carga, ahorrar ancho de banda y aumentar la capacidad de respuesta, ya sea para desarrollar o administrar sitios web o aplicaciones móviles, o codificar y distribuir medios de streaming, software de juegos, actualizaciones de firmware o puntos de conexión de IoT.

## <a name="azure-cdn-features"></a>Características de Azure CDN
  - Funciona **íntegramente** con servicios de Azure, como Web Apps, Media Services, Storage y Cloud Services.
  - **Administración fácil de los picos de tráfico y las cargas pesadas**.
  - **Integración con Akamai y Verizon** para compilar y adaptar la estrategia de entrega de contenido.
  - Aceleración dinámica de sitios.
  - Control del comportamiento de almacenamiento en caché de Azure CDN con **reglas de almacenamiento en caché**.
  - **Compatibilidad con dominios personalizados** HTTPS.
  - **Registros de diagnóstico de Azure** para ver el análisis básico y exportarlo a Blob Storage, Event Hubs o Azure Monitor.
  - Mejora del rendimiento mediante la **compresión de archivos**.
  - **Filtrado geográfico** para restringir el contenido de Azure CDN por país o región.
