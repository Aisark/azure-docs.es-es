# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Content Delivery Network en Azure con Azure Content Delivery Network
[Azure Content Delivery Network](https://azure.microsoft.com/services/cdn/) ofrece a los desarrolladores una solución global para la entrega rápida de contenido de alto ancho de banda (como vídeos, imágenes o archivos estáticos) a los usuarios mediante el almacenamiento en caché de su contenido en cientos de puntos de presencia de todo el mundo. Content Delivery Network también puede acelerar el contenido dinámico, que no se puede almacenar en caché, para reducir los tiempos de carga, ahorrar ancho de banda y aumentar la capacidad de respuesta. Estas ventajas se obtienen al desarrollar o administrar sitios web o aplicaciones móviles, o codificar y distribuir medios de streaming, software de juegos, actualizaciones de firmware o puntos de conexión de IoT.

## <a name="azure-content-delivery-network-features"></a>Características de Azure Content Delivery Network
  - Funciona íntegramente con servicios de Azure, como aplicaciones web, Media Services, Storage y Cloud Services.
  - Administración fácil de los picos de tráfico y las cargas pesadas.
  - Integración con Akamai y Verizon para compilar y adaptar la estrategia de entrega de contenido.
  - Uso de aceleración dinámica de sitios.
  - Control del comportamiento del almacenamiento en caché de Content Delivery Network con reglas de almacenamiento en caché.
  - Obtener compatibilidad con dominios personalizados HTTPS.
  - Consulte los registros de diagnóstico de Azure para ver el análisis básico y exportarlo a Azure Blob Storage, Azure Event Hubs o Azure Monitor.
  - Mejora del rendimiento mediante la compresión de archivos.
  - Use la geofiltración para restringir el contenido de Content Delivery Network por país o región.
