# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>API geoespaciales para agregar mapas, análisis espaciales y soluciones de movilidad con Azure Maps

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) facilita una plataforma de ubicación nativa en la nube y ofrece una cartera de servicios geoespaciales que emplean datos de mapas recientes para proporcionar contexto geográfico preciso a las aplicaciones web y móviles. Los desarrolladores, las empresas y los ISV pueden usar Azure Maps para crear aplicaciones con reconocimiento de ubicación y soluciones de seguimiento de recursos, IoT, movilidad y logística.

## <a name="azure-maps-features"></a>Características de Azure Maps
- **Buscar**: Proporciona datos sobre puntos de interés cercanos, representa ubicaciones en un mapa y usa direcciones de código geográfico (o código geográfico inverso) para obtener las coordenadas de latitud y longitud correspondientes.
- **Tráfico**: mejora el tiempo de viaje y le ayuda a evitar embotellamientos.
- **Mapas**: proporciona mapas claros y fáciles de leer que puede integrar en su solución.
- **Enrutamiento**: Presenta las rutas más cortas o más rápidas disponibles para varios destinos en un momento determinado. El enrutamiento también proporciona rutas y hechos especializados para transeúntes, ciclistas y vehículos comerciales.
- **Zonas horarias**: permite saber fácilmente qué hora es en cualquier parte del mundo.
- **Operaciones espaciales** mejore la inteligencia de ubicación.
- **Geolocalización**: busca el país de una dirección IP.
- **Servicio Data**: carga y almacena datos geoespaciales para usarlos con Spatial Operations o composición de imágenes.
- **Movilidad**: proporciona inteligencia de ubicación en tiempo real en servicios de transporte público cercanos. Se incluyen las paradas, la información de ruta y las estimaciones de tiempo de viaje.
