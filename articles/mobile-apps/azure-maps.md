# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>API geoespaciales para agregar mapas, análisis espaciales y soluciones de movilidad con Azure Maps

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) facilita una plataforma de ubicación nativa en Azure Cloud, ofrece una cartera de servicios geoespaciales que emplean datos de mapas recientes para proporcionar contexto geográfico preciso a las aplicaciones web y móviles. Permite que los desarrolladores, las empresas y los ISV creen aplicaciones con reconocimiento de ubicación y soluciones de seguimiento de recursos, IoT, movilidad y logística.

## <a name="azure-maps-features"></a>Características de Azure Maps
- **Search** proporciona datos sobre puntos de interés (POI) cercanos, representen ubicaciones en un mapa y realicen código geográfico (o código geográfico inverso) de direcciones para obtener las coordenadas de latitud y longitud correspondientes.
- Mejore el tiempo de desplazamiento y evite atascos con **Tráfico**.
- Integre **mapas** claros y fáciles de leer en su solución.
- **Enrutamiento** para presentar la ruta más corta o más rápida disponible (para varios destinos al mismo tiempo) u ofrezca rutas especializadas e información para peatones, ciclistas y vehículos comerciales.
- **Zonas horarias** ofrece los usuarios la posibilidad de saber fácilmente la hora que es en cualquier parte del mundo.
- **Spatial Operations** para mejorar la inteligencia de ubicación.
- **Geolocalización** para buscar el país de una dirección IP.
- **Data Service** para cargar y almacenar datos geoespaciales y usarlos con Spatial Operations o composición de imágenes.
- **Movilidad** ofrece inteligencia de ubicación en tiempo real sobre los servicios de transporte público cercanos, incluidas las paradas, información de las rutas y cálculos del tiempo de los trayectos.
