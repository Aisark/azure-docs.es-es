# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>Servicio en la nube con tecnología de inteligencia artificial con Azure Cognitive Search
[Azure Cognitive Search](https://azure.microsoft.com/services/search/) es una solución en la nube de búsqueda como servicio que proporciona a los desarrolladores las API y las herramientas necesarias para agregar una completa experiencia de búsqueda de datos a través de contenido privado y heterogéneo en aplicaciones web, móviles y empresariales. El código personalizado invoca la ingesta de datos (indexación) para crear y cargar un índice. Por otra parte, el código de aplicación genera solicitudes de consulta y controla las respuestas. La experiencia de búsqueda se define en el cliente mediante el uso de funciones de Azure Cognitive Search, con la ejecución de consultas en un índice persistente que crea, es de su propiedad y almacena en el servicio.

## <a name="azure-cognitive-search-features"></a>Características de Azure Cognitive Search
- **Búsqueda de texto de forma libre** con búsqueda de texto completo, sintaxis de consulta simple y sintaxis de consulta de Lucene.
- **Perfiles de puntuación simples** para modelar la relevancia como una función de los valores de los propios documentos.
- La **búsqueda geográfica** permite a los usuarios explorar los datos según la proximidad de un resultado de búsqueda a una ubicación física.
- **Características de la experiencia de usuario** con autocompletar, sugerencias de búsqueda, ordenación y paginación.
- La **búsqueda cognitiva** para el análisis de texto e imágenes se puede aplicar a una canalización de indexación para extraer información de texto a partir de contenido sin formato.
- **Almacén de conocimiento** para guardar los enriquecimientos creados durante la indexación.
- Conexión a Azure SQL Database, Azure Cosmos DB o Azure Blob Storage como **orígenes de datos** para extraer contenido que se puede buscar en los almacenes de datos principales.
- Uso del **Asistente para la importación de datos** para configurar los indexadores, el **diseñador de índices** para crear un índice y el **Explorador de búsqueda** para probar las consultas y perfeccionar los perfiles de puntuación.

Encuentre una descripción detallada de las [características](/azure/search/search-what-is-azure-search#feature-descriptions) de Azure Cognitive Search.
