# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>Servicio en la nube con tecnología de inteligencia artificial con Azure Cognitive Search
[Azure Cognitive Search](https://azure.microsoft.com/services/search/) es una solución en la nube de búsqueda como servicio que proporciona a los desarrolladores las API y las herramientas necesarias para agregar una completa experiencia de búsqueda de datos a través de contenido privado y heterogéneo en aplicaciones web, móviles y empresariales. El código personalizado invoca la ingesta de datos (indexación) para crear y cargar un índice. Por otra parte, el código de aplicación genera solicitudes de consulta y controla las respuestas. La experiencia de búsqueda se define en el cliente mediante el uso de funciones de Azure Cognitive Search, con la ejecución de consultas en un índice persistente que crea, es de su propiedad y almacena en el servicio.

## <a name="azure-cognitive-search-features"></a>Características de Azure Cognitive Search
- **Búsqueda de texto libre**: proporciona la búsqueda de texto completo, la sintaxis de consulta simple y la sintaxis de consulta de Lucene.
- **Perfiles de puntuación sencillos**: modele la relevancia como función de los valores de los propios documentos.
- **Búsqueda geográfica**: explore los datos a partir de la proximidad de un resultado de búsqueda a una ubicación física.
- **Características de experiencias de usuario**: proporcione características de autocompletar, sugerencias de búsqueda, ordenación y paginación.
- **Cognitive search**: se puede aplicar a una canalización de indexación para extraer información de texto a partir de contenido sin formato para el análisis de texto e imágenes.
- **Knowledge Store**: guarda los enriquecimientos creados durante la indexación.
- **Orígenes de datos**: extraiga contenido utilizable en búsquedas en almacenes de datos cuando esté conectado a Azure SQL Database, Azure Cosmos DB y Azure Blob Storage.
- **Asistente para la importación de datos**: Configura indexadores. 
- **Diseñador de índices**: forma un índice. 
- **Explorador de búsqueda**: prueba las consultas y refina los perfiles de puntuación.

Busque una descripción detallada de las [características](/azure/search/search-what-is-azure-search#feature-descriptions) de Azure Cognitive Search.
