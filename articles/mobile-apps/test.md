# <a name="perform-automated-testing-on-thousands-of-mobile-devices-simultaneously"></a>Realización de pruebas automatizadas en miles de dispositivos móviles simultáneamente
Las pruebas pueden ser desafiantes y cada vez más complejas debido a la fragmentación de dispositivos, complejidad de las aplicaciones, ciclos de versiones rápidos, sesiones cortas y altas expectativas de los usuarios. Debe asegurarse periódicamente de entregar aplicaciones que se han probado exhaustivamente y que proporcionan una excelente experiencia de cliente. Las pruebas no son una tarea fácil y no es un buen uso del tiempo del desarrollador probar manualmente cada versión.

Los desafíos de las pruebas pueden abordarse de forma eficaz. Puede probar aplicaciones móviles en una amplia gama de dispositivos mediante pruebas automatizadas. Puede usar los servicios en la nube para acelerar la publicación de aplicaciones y superar los desafíos de las pruebas que agregan cuellos de botella al desarrollo y la implementación de aplicaciones.

## <a name="importance-of-automated-testing"></a>Importancia de las pruebas automatizadas
- Aumente la cobertura de las pruebas dirigiéndolas a diferentes factores de forma, tipos de conectividad y versiones del sistema operativo.
- Ahorre tiempo, dinero y recursos comprobando la cobertura de distintos dispositivos y sistemas operativos.
- Mejore la precisión evitando pruebas manuales repetitivas y disponga de más tiempo para crear nuevas pruebas automatizadas de software y de tratar características complejas.
- Pruebe continuamente, analice el rendimiento de la aplicación, busque y solucione problemas antes de que sus usuarios se topen con ellos, y publique rápidamente.
- Ofrezca aplicaciones de alta calidad asegurándose de que los clientes de una amplia gama de dispositivos obtienen la mejor experiencia.

## <a name="visual-studio-app-center"></a>Visual Studio App Center
[App Center Test](/appcenter/test-cloud/) es un servicio basado en la nube que puede usar para ejecutar pruebas de la interfaz de usuario automatizadas en miles de dispositivos reales en la nube a través de Android e iOS para aplicaciones móviles nativas e híbridas. Con más de 2000 dispositivos, puede escribir scripts de prueba que se pueden ejecutar automáticamente para facilitar las pruebas.

**Características principales**
   - Pruebas que se ejecutan en dispositivos reales sin rooting o jailbreaking, lo que proporciona la mejor aproximación de los entornos de los usuarios.
   - Se realizan capturas de pantalla automáticamente y se adjuntan a cada paso de prueba. Se pueden ver en paralelo para todos los dispositivos de destino de una serie de pruebas.
   - Registros del dispositivo para todos los dispositivos de destino de una serie de pruebas y seguimiento de la pila para los pasos de prueba con errores.
   - Series de pruebas simultáneas en varios dispositivos a la vez para acelerar el tiempo de actividad para el conjunto de resultados.
   - Compatibilidad con marcos de pruebas conocidos como Appium (Java con JUnit) para Android e iOS, Espresso (Java) para Android, Calabash para Android e iOS, Xamarin.UITest para Android e iOS y XCUITest para iOS.
   - Compatibilidad con las plataformas iOS, Android, Xamarin y React Native.
