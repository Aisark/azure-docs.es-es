# <a name="update-the-azure-backup-central-reporting-content-pack"></a>Actualización del paquete de contenido de informes central de Azure Backup 

El [paquete de contenido de Azure Backup](https://docs.microsoft.com/azure/backup/backup-azure-configure-reports#view-reports-in-power-bi) se puede usar para ver informes sobre copias de seguridad centrales. El paquete de contenido se actualiza con regularidad para agregar más características y para corregir errores. En este artículo se explica cómo actualizar el paquete de contenido. También se indica cómo posponer la actualización y ver las actualizaciones realizadas con el tiempo.

## <a name="get-updates-to-the-content-pack"></a>Obtención de actualizaciones para el paquete de contenido

### <a name="get-the-updated-content-pack"></a>Obtención del paquete de contenido actualizado
Si no ha hecho cambios en la copia del paquete de contenido, se actualizará automáticamente. Cuando cambie el paquete de contenido, recibirá una notificación en Power BI y una notificación por correo electrónico. Puede obtener el paquete de contenido actualizado según su preferencia. 

### <a name="postpone-the-update"></a>Posponer la actualización
El procedimiento recomendado consiste en importar el paquete de contenido en un [área de trabajo personalizada](https://youtu.be/26zyOtyHPJM?t=1m57s). Ahora puede editar los informes.
Como se mencionó anteriormente, si se cambia el paquete de contenido, verá una notificación en Power BI. Puede elegir obtener el paquete de contenido más tarde. 

## <a name="coming-soon"></a>Próximamente
   
El paquete de contenido de Azure Backup se actualiza para admitir más cargas de trabajo. Las cargas de trabajo incluyen Azure SQL Database para la copia de seguridad de VM de IaaS y System Center Data Protection Manager. Esta compatibilidad se agrega a la compatibilidad actual para Azure Backup y las copias de seguridad de máquinas virtuales de Azure. Esta compatibilidad significa que puede ver y analizar todos los datos de las copias de seguridad en un lugar centralizado. [También se pueden personalizar los informes](https://youtu.be/26zyOtyHPJM) para satisfacer las necesidades de su organización.

Los informes preconfigurados que vienen con el paquete de contenido de Azure Backup están cambiando. Los cambios permiten que los informes sean más significativos en las cargas de trabajo. Puede echar un vistazo al próximo conjunto de informes aquí.
