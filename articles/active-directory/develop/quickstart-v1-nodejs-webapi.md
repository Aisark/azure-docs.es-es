# <a name="quickstart-secure-a-web-api-with-azure-active-directory"></a>Inicio rápido: Proteger una API web con Azure Active Directory

[!INCLUDE [active-directory-develop-applies-v1](../../../includes/active-directory-develop-applies-v1.md)]

La [Plataforma de identidad de Microsoft](v2-overview.md) es una evolución de la plataforma para desarrolladores de Azure Active Directory (Azure AD). Permite a los desarrolladores compilar aplicaciones que inicien sesión en todas las identidades de Microsoft, obtener tokens para llamar a las API de Microsoft como Microsoft Graph, o a otras API que los desarrolladores hayan creado.

## <a name="next-steps"></a>Pasos siguientes

Consulta el [ejemplo web de OIDC en Node.js para Azure Active Directory](https://github.com/azureadquickstarts/appmodelv2-webapp-openidconnect-nodejs) en el que se muestra cómo configurar la autenticación de OpenId Connect con Azure AD v2.0 en una aplicación web creada mediante Node.js con Express. Está diseñado para ejecutarse en cualquier plataforma.
