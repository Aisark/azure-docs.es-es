# <a name="what-is-the-my-apps-preview-portal"></a>¿Qué es el portal Aplicaciones (versión preliminar)?

Puede usar la cuenta profesional o educativa con el portal web **Aplicaciones** para ver, organizar e iniciar muchas de las aplicaciones basadas en la nube de la organización. Si no tiene acceso al portal **Aplicaciones**, póngase en contacto con el departamento de soporte técnico de la organización para obtener permiso.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-and-workspaces.md)]

>[!Important]
>Este contenido está diseñado para usuarios. Si es administrador, puede encontrar más información acerca de cómo configurar y administrar aplicaciones basadas en la nube en la [Documentación sobre la administración de aplicaciones](https://docs.microsoft.com/azure/active-directory/manage-apps/access-panel-workspaces).
