# <a name="access-and-start-apps-from-the-my-apps-preview-portal"></a>Acceso e inicio de aplicaciones desde el portal Aplicaciones (versión preliminar)

Puede usar la cuenta profesional o educativa con el portal web **Aplicaciones** (versión preliminar) para ver, organizar e iniciar muchas de las aplicaciones basadas en la nube de la organización. Si no tiene acceso al portal **Aplicaciones**, póngase en contacto con el departamento de soporte técnico de la organización para obtener permiso.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-and-workspaces.md)]

>[!Important]
>Este contenido está diseñado para usuarios. Si es administrador, puede encontrar más información acerca de cómo configurar y administrar aplicaciones basadas en la nube en la [Documentación sobre la administración de aplicaciones](https://docs.microsoft.com/azure/active-directory/manage-apps/access-panel-workspaces).

## <a name="supported-browsers"></a>Exploradores compatibles

Puede obtener acceso al portal **Mis aplicaciones** desde cualquiera de los exploradores web siguientes:

- Google Chrome

- Mozilla Firefox, versión 26.0 o posterior

- Microsoft Edge

- Internet Explorer, versión 11 (compatibilidad limitada)

## <a name="access-the-my-apps-portal-by-device-type"></a>Acceso al portal Aplicaciones por tipo de dispositivo

Puede acceder al portal **Aplicaciones** y usarlo en su equipo profesional o educativo, desde un explorador administrado por Intune o desde un dispositivo móvil iOS o Android.

![Página de aplicaciones en el portal Mis aplicaciones](media/my-apps-portal-preview/my-apps-portal-preview.png)

### <a name="access-the-my-apps-portal-on-a-computer"></a>Acceso al portal Aplicaciones en un equipo

Si tiene permiso para obtener acceso y usar aplicaciones basadas en la nube de su organización, puede obtenerlas a través del portal **Mis aplicaciones**.

1. Inicie sesión en la cuenta profesional o educativa en su equipo.

2. Abra un explorador web compatible y vaya a https://myapps.microsoft.com, o bien use el vínculo proporcionado por la organización. Por ejemplo, es posible que se le redirija para usar una página personalizada para la organización, como `https://myapps.microsoft.com/contoso.com`.

    Aparecerá la página **Aplicaciones**, que muestra todas las aplicaciones basadas en la nube disponibles para su uso y propiedad de su organización.

### <a name="access-the-my-apps-portal-from-the-intune-managed-browser-app"></a>Acceso al portal Aplicaciones desde la aplicación Intune Managed Browser

Vea y use las aplicaciones de la organización desde la aplicación Intune Managed Browser en los dispositivos iOS o Android.

1. En el dispositivo móvil, descargue e instale la aplicación Intune Managed Browser desde la [App Store de Apple](https://itunes.apple.com/us/app/microsoft-intune-managed-browser/id943264951?mt=8) o la [Google Play Store](https://play.google.com/store/apps/details?id=com.microsoft.intune.mam.managedbrowser).

2. Abra la aplicación Intune Managed Browser y vaya a https://myapps.microsoft.com o use el vínculo proporcionado por su organización. Por ejemplo, quizá se le redirija a una página personalizada para la organización, como https://myapps.microsoft.com/contoso.com.

    Aparecerá la página **Aplicaciones**, que muestra todas las aplicaciones basadas en la nube disponibles para su uso y propiedad de su organización.

### <a name="access-the-my-apps-portal-on-an-ios-device"></a>Acceso al portal Aplicaciones en un dispositivo iOS

Vea y use el portal **Mis aplicaciones** desde un dispositivo iPhone o iPad con la versión iOS 7 o posterior. También puede instalar la [aplicación móvil Mis aplicaciones](https://itunes.apple.com/us/app/my-apps-azure-active-directory/id824048653?mt=8) para acceder a aplicaciones de su organización en sus dispositivos iOS.

1. En su dispositivo móvil, inicie una aplicación de explorador web, como Safari.

2. Vaya a https://myapps.microsoft.com o use el vínculo proporcionado por su organización. Por ejemplo, quizá se le redirija a una página personalizada para la organización, como https://myapps.microsoft.com/contoso.com.

    Aparecerá la página **Aplicaciones**, que muestra todas las aplicaciones basadas en la nube disponibles para su uso y propiedad de su organización.

### <a name="access-the-my-apps-portal-on-an-android-device"></a>Acceso al portal Aplicaciones en un dispositivo Android

Vea y use el portal **Mis aplicaciones** en un dispositivo Android.

1. En su dispositivo móvil, inicie una aplicación de explorador web, como Google Chrome.

2. Vaya a https://myapps.microsoft.com o use el vínculo proporcionado por su organización. Por ejemplo, quizá se le redirija a una página personalizada para la organización, como https://myapps.microsoft.com/contoso.com.

    Aparecerá la página **Aplicaciones**, que muestra todas las aplicaciones basadas en la nube disponibles para su uso y propiedad de su organización.

## <a name="start-a-cloud-based-app"></a>Inicio de una aplicación basada en la nube

Puede iniciar cualquiera de las aplicaciones basadas en la nube disponibles desde el portal **Aplicaciones**. Solo verá las aplicaciones para las que tiene permiso de uso.

- Desde la página **Aplicaciones**, seleccione la aplicación que desea empezar a usar.

    Se abrirá una nueva página para la aplicación, en la que podrá iniciar sesión (si es necesario) o empezar a usar la aplicación.

## <a name="next-steps"></a>Pasos siguientes

Cuando esté en el portal **Aplicaciones**, podrá:

- Organizar las aplicaciones en las distintas categorías creadas y proporcionadas por la organización. Para más información, vea [Acceso a las áreas de trabajo y uso de las mismas en el portal Aplicaciones (versión preliminar)](my-applications-portal-workspaces.md).

- Revisar, actualizar o revocar los permisos concedidos a las aplicaciones. Para más información, vea [Edición o revocación de permisos de aplicación en el portal Aplicaciones (versión preliminar)](my-applications-portal-permissions-saved-accounts.md).

>[!Note]
>Si no ve ningún área de trabajo o categoría disponible, significa que el administrador no ha configurado ni compartido ninguna con usted. Póngase en contacto con el departamento de soporte técnico de la organización para obtener ayuda adicional o permisos para ver las áreas de trabajo compartidas.
