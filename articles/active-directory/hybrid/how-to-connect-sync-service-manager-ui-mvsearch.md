# <a name="sync-service-manager-metaverse-search"></a>Búsqueda de metaverso de Synchronization Service Manager

![Sync Service Manager](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

La pestaña Búsqueda de metaverso resulta útil para solucionar problemas relacionados con datos. En la mitad superior, puede crear una consulta basada en una combinación de atributos. Cuando le satisfaga la consulta, haga clic en **Búsqueda**. El resultado aparecerá en la cuadrícula inferior. Puede seleccionar qué columnas deben estar visibles en **Configuración de columnas**.

En los resultados de búsqueda, seleccione un objeto y **Propiedades** para ver las [propiedades del objeto de metaverso](tshoot-connect-object-not-syncing.md#metaverse-object-properties).
