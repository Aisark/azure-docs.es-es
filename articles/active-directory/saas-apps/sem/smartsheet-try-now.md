# <a name="smartsheet-and-azure-active-directory"></a>Smartsheet y Azure Active Directory

Acelere la adopción de Smartsheet en su organización y aumente la adopción por parte de los usuarios con nuestra integración pregenerada. Conecte Smartsheet a Azure Active Directory (Azure AD) para permitir que los usuarios colaboren rápidamente.

![Muestra el logotipo de la aplicación Smartsheet](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [Pruebe Smartsheet con Azure AD](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>¿Qué puede hacer con Smartsheet y Azure AD?

- **Inicio de sesión único sencillo**: elimine varios nombres de usuario y contraseñas y permita que los usuarios inicien sesión en Smartsheet con sus cuentas de Azure AD para simplificar el acceso.
- **Aprovisionamiento automático de usuarios**: automatice la creación, el mantenimiento y la eliminación de cuentas de usuario en Smartsheet cuando los usuarios se unen, se mueven o se van de su organización.
- **Proteja, controle y supervise el acceso**: proteja el acceso a Smartsheet mediante la aplicación de directivas de acceso condicional, controle el acceso por medio de revisiones de acceso y supervise el acceso con los registros de inicio de sesión.

## <a name="get-started"></a>Primeros pasos

Use nuestras sencillas guías de configuración para configurar el [aprovisionamiento de usuarios](https://aka.ms/SmartsheetProvisioningDoc) y el [inicio de sesión único](https://aka.ms/SmartsheetSSODoc) para Smartsheet.
