# <a name="box-and-azure-active-directory"></a>Box y Azure Active Directory

Acelere la adopción de Box en toda la organización y aumente la adopción por parte de los usuarios con nuestra integración pregenerada. Conecte Box a Azure Active Directory (Azure AD) para permitir que los usuarios colaboren rápidamente.

![Muestra el logotipo de la aplicación Box](./media/box.png)

> [!div class="nextstepaction"]
> [Pruebe Box con Azure AD](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>¿Qué puede hacer con Box y Azure AD?

- **Inicio de sesión único sencillo**: elimine varios nombres de usuario y contraseñas y permita que los usuarios inicien sesión en Box con sus cuentas de Azure AD para simplificar el acceso.
- **Aprovisionamiento automático de usuarios**: automatice la creación, el mantenimiento y la eliminación de cuentas de usuario en Box cuando los usuarios se unen, se mueven o se van de su organización.
- **Proteja, controle y supervise el acceso**: proteja el acceso a Box mediante la aplicación de directivas de acceso condicional, controle el acceso por medio de revisiones de acceso y supervise el acceso con los registros de inicio de sesión.

## <a name="get-started"></a>Primeros pasos

Use nuestras sencillas guías de configuración para configurar el [aprovisionamiento de usuarios](https://aka.ms/BoxSSODoc) y el [inicio de sesión único](https://aka.ms/BoxProvisioningDoc) para Box.
