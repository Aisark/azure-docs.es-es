# <a name="workday-and-azure-active-directory"></a>Workday y Azure Active Directory

Conecte Workday con Azure Active Directory (Azure AD) mediante nuestra integración preconfigurada y aprovisione o desaprovisione automáticamente las cuentas de usuario cuando los empleados se unan, se muevan o salgan.

![Muestra el logotipo de la aplicación Workday](./media/workday.png)

> [!div class="nextstepaction"]
> [Probar Workday con Azure AD](https://aka.ms/WorkdayTryItNow)

## <a name="what-can-you-do-with-workday-and-azure-ad"></a>¿Qué puede hacer con Workday y Azure AD?

- **Aprovisionamiento automático de usuarios**: asigne aplicaciones automáticamente a los empleados en función de su grupo, rol u otros atributos en Workday.
- **Sincronización de los perfiles de empleado**: asegúrese de que los datos de los empleados permanecen actualizados entre Azure AD y Workday.
- **Simplificación de la incorporación y la retirada**: aprovisione automáticamente las cuentas de usuario de los nuevos empleados de forma que puedan ser productivos desde el primer día y desaprovisione automáticamente las cuentas de usuario cuando de los empleados que se van para reducir los riesgos de seguridad.
- **Inicio de sesión único sencillo**: elimine el nombre de usuario y las contraseñas y permita que los usuarios inicien sesión con sus cuentas organizativas hospedadas en Azure AD.

## <a name="get-started"></a>Primeros pasos

Configure el [aprovisionamiento de usuarios](https://aka.ms/WorkdayProvisioning) y el [inicio de sesión único](https://aka.ms/WorkdaySSODoc) para Workday con nuestras sencillas guías de configuración.
