# <a name="servicenow-and-azure-active-directory"></a>ServiceNow y Azure Active Directory

Acelere la adopción de ServiceNow en la organización y aumente la adopción por parte de los usuarios con nuestra integración pregenerada. Conecte ServiceNow a Azure Active Directory (Azure AD) y optimice su administración de servicios de TI.

![Muestra el logotipo de la aplicación ServiceNow](./media/servicenow.png)

> [!div class="nextstepaction"]
> [Pruebe ServiceNow con Azure AD](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>¿Qué puede hacer con ServiceNow y Azure AD?

- **Inicio de sesión único sencillo**: elimine varios nombres de usuario y contraseñas y permita que los usuarios inicien sesión en ServiceNow con sus cuentas de Azure AD para simplificar el acceso.
- **Aprovisionamiento automático de usuarios**: automatice la creación, el mantenimiento y la eliminación de cuentas de usuario en ServiceNow cuando los usuarios se unen, se mueven o se van de su organización.
- **Proteja, controle y supervise el acceso**: proteja el acceso a ServiceNow mediante la aplicación de directivas de acceso condicional, controle el acceso por medio de revisiones de acceso y supervise el acceso con los registros de inicio de sesión.

## <a name="get-started"></a>Primeros pasos

Use nuestras sencillas guías de configuración para configurar el [aprovisionamiento de usuarios](https://aka.ms/ServiceNowProvisioning) y el [inicio de sesión único](https://aka.ms/ServiceNowSSO) para ServiceNow.
