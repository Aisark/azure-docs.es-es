# <a name="micro-focus-devtest-environments-on-azure"></a>Entornos de desarrollo y pruebas de Micro Ficus en Azure

Comenzar a usar Azure con un entorno de desarrollo y pruebas es una excelente manera de potenciar sus labores de modernización. Las aplicaciones de sistema central que dependen de sistemas heredados, como IBM CICS (Customer Information Control System) e IMS (Information Management System) se pueden rehospedar en Azure mediante soluciones de asociados.

Micro Focus colabora con Microsoft para ofrecer atractivas herramientas de migración y rehospedaje de sistemas centrales. Puede ejecutar aplicaciones de sistema central en escenarios de desarrollo, pruebas y producción en Azure y en la plataforma Microsoft Windows.

Micro Focus ofrece soluciones de rehospedaje de desarrollo y pruebas que amplían las aplicaciones empresariales existentes de COBOL y PL/I a Azure.
