# <a name="micro-focus-devtest-environments-on-azure"></a>Entornos de desarrollo y pruebas de foco microcargas en Azure

Introducción a un entorno de desarrollo y pruebas en Azure es una excelente manera de aumentar sus esfuerzos de modernización. Aplicaciones de grandes sistemas que dependen de los sistemas heredados como (Customer Information Control System) de IBM CICS e IMS (sistema de administración de información) pueden rehospedarse en Azure con las soluciones de asociados.

Micro Focus colabora con Microsoft para ofrecer las herramientas de migración y rehospedaje de mainframe atractiva. Puede ejecutar aplicaciones de gran sistema en desarrollo, pruebas y escenarios de producción en Azure y la plataforma Microsoft Windows.

Micro Focus ofrece rehospedaje de desarrollo y prueba rehospedaje soluciones que amplían COBOL existente y PL / me aplicaciones empresariales en Azure.
