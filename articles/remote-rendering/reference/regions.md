# <a name="regions"></a>Regions

En esta página se enumeran las regiones disponibles para utilizarse con Azure Remote Rendering. Para obtener los mejores resultados, use siempre la región más cercana a su ubicación física.

| Nombre | Region | URL |
|-----------|:-----------|:-----------|
| Oeste de EE. UU. 2 | westus2 | `https://remoterendering.westus2.mixedreality.azure.com` |
| Este de EE. UU. | estado | `https://remoterendering.eastus.mixedreality.azure.com` |
| Oeste de Europa | westeurope | `https://remoterendering.westeurope.mixedreality.azure.com` |
| Sudeste de Asia | southeastasia | `https://remoterendering.southeastasia.mixedreality.azure.com` |
