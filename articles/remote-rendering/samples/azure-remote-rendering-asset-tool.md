# <a name="azure-remote-rendering-asset-tool-arrt"></a>Herramienta de recursos Azure Remote Rendering (ARRT)

La herramienta de recursos Azure Remote Rendering (ARRT) es una aplicación de escritorio de código abierto desarrollada en C++ /Qt.

![ARRT](./media/azure-remote-rendering-asset-tool.png "Captura de pantalla de ARRT")

## <a name="features"></a>Características

La aplicación se puede usar para:

* Cargar un modelo 3D.
* Controlar la conversión de modelos.
* Crear y administrar una sesión de representación remota.
* Cargar un modelo 3D.
* Obtener una vista previa de un modelo 3D
* Modificar sus materiales

También se puede usar como ejemplo de código abierto para aprender a implementar un front-end para el SDK de C++ de Azure Remote Rendering, mediante la biblioteca de cliente de Azure Storage para administrar la conversión de modelos 3D.

## <a name="source-repository"></a>Repositorio de origen

Puede encontrar el código fuente y la documentación en el [repositorio ARRT de GitHub](https://github.com/Azure/azure-remote-rendering-asset-tool).
