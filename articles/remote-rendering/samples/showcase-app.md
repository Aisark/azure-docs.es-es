# <a name="unity-showcase-application"></a>Aplicación de presentación de Unity

La aplicación de presentación Unity es una aplicación de ejemplo basada en Unity para mostrar las características de Azure Remote Rendering. Puede ejecutarse tanto en el editor de Unity como en Hololens 2.

![Aplicación de presentación](./media/showcase-app.png "Aplicación de presentación")

## <a name="source-repository"></a>Repositorio de origen

El código fuente de esta aplicación junto con las instrucciones de compilación e implementación se encuentran en el [repositorio de ejemplos de ARR](https://github.com/Azure/azure-remote-rendering/tree/master/Unity/Showcase).
