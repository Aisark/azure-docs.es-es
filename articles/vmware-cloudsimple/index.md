# <a name="azure-vmware-solution-by-cloudsimple"></a>Azure VMware Solution by CloudSimple

Bienvenido a Azure VMware Solution by CloudSimple. Aquí encontrará más información sobre cómo usar CloudSimple y cómo habilitar soluciones en el servicio. Busque los artículos o examine el índice.

- Consulte la [introducción](cloudsimple-vmware-solutions-overview.md) para más información sobre Azure VMware Solution by CloudSimple.
- Seleccione [CloudSimple private cloud overview](cloudsimple-private-cloud.md) (Introducción a la nube privada de CloudSimple) para más información sobre la solución de nube privada de CloudSimple.
- Seleccione [Nodes](cloudsimple-node.md) (Nodos) para más información sobre los nodos de CloudSimple.
- Seleccione [Initialize and provision capacity](quickstart-create-cloudsimple-service.md) (Inicialización y aprovisionamiento de capacidad) para empezar a usar Azure VMware Solution by CloudSimple.

| Conceptos de un vistazo     | Recurso                    | Propósito                                                                       | Administración         |
|--------------------------|-----------------------------|-------------------------------------------------------------------------------|--------------------|
| **Creación de servicios**     | Servicio CloudSimple         | Configuración de la red perimetral del servicio CloudSimple                                    | Portal de Azure       |
|                          | Nodo de CloudSimple            | Aprovisionamiento de la capacidad que se usará en el servicio CloudSimple                          | Portal de Azure       |
| **Consumo de servicio**  | Nube privada               | Creación de entornos de VMware                                                    | Portal de CloudSimple |
|                          | VLAN/Subred                 | Creación de redes subyacentes para el consumo de VMware                               | Portal de CloudSimple |
|                          | Tabla de firewall              | Configuración de reglas de seguridad verticales de arriba abajo y horizontales de derecha a izquierda en las redes subyacentes | Portal de CloudSimple |
|                          | Dirección IP pública           | Obtención de direcciones públicas para las cargas de trabajo de VMware                           | Portal de CloudSimple |
|                          | Conexión de Azure            | Establecer conectividad con redes locales o virtuales de Azure         | Portal de CloudSimple |
| **Administración unificada**   | Máquina virtual de CloudSimple | Administración de máquinas virtuales de VMware en Azure Portal                                | Portal de Azure       |
