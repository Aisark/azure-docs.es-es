# <a name="what-is-vmware-solution-on-azure-by-cloudsimple"></a>¿Qué es la solución de VMware en Azure por CloudSimple

**Solución de VMware en Azure por CloudSimple** es un servicio totalmente administrado que le permite ejecutar la plataforma de VMware en Azure. Esta solución incluye vSphere, vCenter, vSAN, NSX-T y herramientas correspondientes.
El entorno de VMware se ejecuta nativamente en infraestructura de Azure sin sistema operativo, en las ubicaciones de la nube de Azure. El servicio incluye todas las características necesarias para consumir las plataformas de VMware de manera eficaz y segura.

![Solución de VMware en Azure mediante la introducción de CloudSimple](media/azure-vmware-solution-by-cloudsimple.png)

## <a name="features"></a>Características

* El aprovisionamiento de petición autoservicio de entornos de VMware en la nube. Capacidad de agregar y quitar la capacidad bajo demanda
* Implementación de la plataforma de VMware, actualización, copia de seguridad de plano de administración, mantenimiento y capacidad de supervisión, alertas, solución de problemas y corrección.
* Subpuesto servicios de red necesarios para habilitar VMware, incluidos administración de reglas de firewall y de servicios de L2/L3.
* Tipo de borde servicios de red, como VPN, las direcciones IP y puertas de enlace de Internet. Estos servicios edge ejecuten en Azure y llevan el correspondiente seguridad y protección contra DDoS de Azure.
* Reserva de capacidad para reducir los costos.
* Conectividad de alta velocidad y baja latencia en Azure y locales.
* Arquitecturas de soluciones para los clientes consumir servicios de Azure de forma integrada, aprovechar las ventajas de esta arquitectura única de "Nube de VMware en una nube pública". Estos servicios de Azure incluyen Azure AD, almacenamiento, las puertas de enlace de aplicaciones y otros.
* Infraestructura está dedicado totalmente a usted y está aislada físicamente de la infraestructura de otros clientes.
* Las características de administración tales administración de actividades, uso, facturación y medición y administración de usuarios.
* asistencia al cliente de 24 x 7.

## <a name="benefits"></a>Ventajas

* **Continuidad operativa:** CloudSimple ofrece acceso nativo a plataformas de VMware. La arquitectura CloudSimple es compatible con el contenido existente:
  * APLICACIONES
  * Operaciones
  * Redes
  * Seguridad
  * Copia de seguridad
  * Recuperación ante desastres
  * Auditoría
  * Herramientas de compatibilidad
  * Procesos
* **Ninguna operación de reciclaje:** Compatibilidad de la plataforma VMware permite usar los conocimientos y habilidades existentes.
* **Agilidad de la infraestructura:** Ya no tienen que predecir todas sus necesidades de capacidad y, a continuación, terminará con capacidad desperdiciada o situaciones de escasez de infraestructura. CloudSimple se entrega como un servicio en la nube, y puede agregar o reducir la capacidad en cualquier momento
* **Seguridad:** Acceso al entorno CloudSimple a través de Azure proporciona protección contra DDoS integrada y supervisión de seguridad.
* **Menor costo:** La plataforma CloudSimple alta se ha diseñado y proporciona altos niveles de automatización, la eficacia operativa y las economías de escala. Además, CloudSimple publica las arquitecturas de soluciones que aprovechen la presencia de VMware en una nube pública para reducir los costos. Algunos ejemplos incluyen la copia de seguridad en almacenamiento de Azure, puerta de enlace de aplicaciones de Azure AD, equilibrador de carga y otros.
* **Una nueva plataforma híbrida:** El servicio permite el acceso de alta velocidad y baja latencia al resto de Azure. Además, la administración de CloudSimple permite una administración unificada de máquinas virtuales de VMware y el resto de Azure con la misma interfaz de usuario y API. Los equipos de desarrollo pueden aprovechar las ventajas de plataformas públicas y privadas de un modo integrado y coherente.
* **Infraestructura de supervisión, solución de problemas y soporte técnico:** CloudSimple funciona su infraestructura subyacente como un servicio. Producirá un error en hardware se reemplazará automáticamente. Le permite centrarse en el consumo mientras CloudSimple garantiza que el entorno se ejecuta sin problemas.
* **Compatibilidad con la directiva:** Mantenga sus herramientas basadas en VMware, los procedimientos de seguridad, procedimientos recomendados de auditoría y certificaciones de cumplimiento.

## <a name="scenarios"></a>Escenarios

* **Retirada de centro de datos o la migración:** Obtener capacidad adicional cuando se alcanzan los límites de su centro de datos existente o actualiza el hardware. Es fácil para agregar la capacidad necesaria en la nube, y eliminar los problemas de administración de actualizaciones de hardware. Reducir el riesgo y el costo de las migraciones de nube en comparación con las conversiones que requieren mucho tiempo o mejorarlo. Use herramientas conocidas de VMware y las habilidades para acelerar las migraciones de nube. Una vez en la nube, use servicios de Azure para modernizar las aplicaciones a su ritmo.
* **Expanda bajo demanda:** Expandir a la nube para satisfacer las necesidades no anticipadas, como nuevos entornos de desarrollo o picos estacionales de capacidad. Fácilmente puede crear la nueva capacidad bajo demanda y manténgala solo siempre que lo necesite. Reducir la inversión inicial, aceleran la velocidad de aprovisionamiento y reduzca la complejidad con la misma arquitectura y las directivas en un entorno local y la nube.
* **Recuperación ante desastres y escritorios virtuales en la nube de Azure:** Establecer el acceso remoto a datos, aplicaciones y escritorios en la nube de Azure. Con conexiones de gran ancho de banda, cargar o descargar datos rápida para recuperarse de incidentes. Dar de baja latencia redes rápida respuesta el tiempo que los usuarios esperan de una aplicación de escritorio. Con CloudSimple, es fácil replicar todas las directivas y las redes en la nube mediante el portal de CloudSimple y herramientas conocidas de VMware. La facilidad de replicación y recuperación reduce en gran medida el esfuerzo y el riesgo de crear y administrar implementaciones de VDI y de recuperación ante desastres.
* **Aplicaciones de alto rendimiento y las bases de datos:** CloudSimple proporciona una arquitectura de hiperconvergido diseñada para ejecutar sus cargas de trabajo más exigentes de VMware. Ejecute Oracle, Microsoft SQL server, los sistemas de software intermedio y las bases de datos NOSQL de alto rendimiento. Experimente la nube como su propio centro de datos con conexiones de red de alta velocidad 25 Gbps que le permiten ejecutar aplicaciones híbridas que abarcan de forma local VMware en Azure y Azure las cargas de trabajo privadas sin comprometer el rendimiento.
* **Híbrida verdadera:** Unificar DevOps en Azure y VMware. Optimizar la administración de VMware para servicios de Azure y soluciones que se pueden aplicar en todas las cargas de trabajo. Obtener acceso a servicios de nube pública sin tener que ampliar su centro de datos o rediseñar sus aplicaciones. Centralizar las identidades, las directivas de control de acceso de registro y supervisión de aplicaciones de VMware en Azure.
