# <a name="activity-management-overview"></a>Introducción a la administración de actividad

CloudSimple realiza un seguimiento de todas las actividades que pueden afectar al funcionamiento de sus nubes privadas. Estas actividades incluidas las alertas, eventos, tareas y actividad de auditoría. El [páginas de la actividad](https://docs.azure.cloudsimple.com/activity/) resumir toda la actividad actual y le permiten explorar en profundidad para obtener más detalles.

## <a name="events"></a>Eventos

Eventos de realizar un seguimiento de usuario y la actividad del sistema en el portal de CloudSimple.  Los eventos muestran la actividad asociada a un recurso específico y la gravedad del impacto.  Puede ver los eventos desde el portal de CloudSimple.

## <a name="alerts"></a>Alertas

Las alertas son notificaciones de cualquier actividad significativa en el entorno CloudSimple.  Eventos, lo que afectar al acceso de usuario o de facturación, se muestran como alertas.  Puede confirmar las alertas activadas desde el portal de CloudSimple.

## <a name="tasks"></a>Tareas

Tareas de realizar un seguimiento de cualquier operación de usuario, que tarda más de 30 segundos en completarse.  Puede supervisar el progreso de una tarea desde el portal de CloudSimple.  Completar tareas muestran el tiempo total necesario completar la tarea.

## <a name="audit"></a>Auditoría

El usuario ha realizado de operaciones de seguimiento de registros de auditoría.  Los registros contienen los parámetros que el usuario proporcionó para la operación.  Puede usar los registros de auditoría para supervisar la actividad de usuario para todos los usuarios.
