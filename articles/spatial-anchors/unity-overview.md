# <a name="building-in-unity-with-azure-spatial-anchors"></a>Compilación en Unity con Azure Spatial Anchors

Los desarrolladores pueden elegir Unity para crear e implementar aplicaciones de realidad mixta que utilizan Azure Spatial Anchors. Se admiten las siguientes plataformas:

**Unity para HoloLens**

[Inicio rápido: Creación de una aplicación HoloLens en Unity que use Azure Spatial Anchors](https://docs.microsoft.com/azure/spatial-anchors/quickstarts/get-started-unity-hololens)

**Unity para Android**

[Inicio rápido: Creación de una aplicación Android en Unity que use Azure Spatial Anchors](https://docs.microsoft.com/azure/spatial-anchors/quickstarts/get-started-unity-android)

**Unity para iOS**

[Inicio rápido: Creación de una aplicación iOS en Unity que use Azure Spatial Anchors](https://docs.microsoft.com/azure/spatial-anchors/quickstarts/get-started-unity-ios)
