# <a name="azure-synapse-analytics-business-intelligence-partners"></a>Asociados de inteligencia empresarial de Azure Synapse Analytics

En este artículo se destacan las empresas asociadas de Microsoft con soluciones de inteligencia empresarial oficiales (BI) compatibles con Azure SQL Data Warehouse.
