## <a name="languages---community-supported"></a>Lenguajes: admitidos por la comunidad

Hay una serie de SDK de Application Insights admitidos por la comunidad, muchos de los cuales creó originalmente Microsoft. Los SDK admitidos por la comunidad no los mantiene Microsoft oficialmente y no podemos ofrecer soporte técnico para ningún SDK que no se encuentre en la lista de admitidos. Estos SDK se consideran experimentales y no se recomiendan para producción.
