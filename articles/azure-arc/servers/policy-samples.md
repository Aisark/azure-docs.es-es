# <a name="azure-policy-built-in-definitions-for-azure-arc-for-servers"></a>Definiciones integradas de Azure Policy en Azure Arc para servidores

Esta página es un índice de las definiciones de directivas integradas de [Azure Policy](../../governance/policy/overview.md) en Azure Arc para servidores. Puede encontrar elementos integrados adicionales de Azure Policy para otros servicios en [Definiciones de elementos integrados de Azure Policy](../../governance/policy/samples/built-in-policies.md).

El nombre de cada definición de directiva integrada se vincula a la definición de directiva en Azure Portal. Use el vínculo de la columna **Versión** para ver el origen en el [repositorio de GitHub de Azure Policy](https://github.com/Azure/azure-policy).
