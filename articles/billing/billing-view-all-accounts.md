# <a name="view-your-billing-accounts-in-azure-portal"></a>Visualización de las cuentas de facturación en Azure Portal  

Se crea una cuenta de facturación por cada contrato que firme con Microsoft para usar Azure. Puede utilizar la cuenta de facturación para administrar las facturas y métodos de pago, y para hacer un seguimiento de los costos de los servicios de Azure. Puede tener acceso a varias cuentas de facturación. Por ejemplo, es posible que te hayas registrado en Azure para tus proyectos personales. También se puede tener acceso a Azure a través del Contrato Enterprise de la organización o del contrato de cliente de Microsoft. Para cada uno de estos escenarios, se dispondría de una cuenta de facturación independiente.

Azure Portal admite actualmente los siguientes tipos de cuentas de facturación:

- **Microsoft Online Services Program**: Se crea una cuenta de facturación para Microsoft Online Services Program cuando se registra en el sitio web de Azure. Por ejemplo, cuando se registra en una [cuenta gratuita de Azure](https://azure.microsoft.com/offers/ms-azr-0044p/), en una [oferta de Pago por uso de Azure](https://azure.microsoft.com/offers/ms-azr-0003p/) o como un [suscriptor de Visual studio](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/).

- **Contrato Enterprise**: Se crea una cuenta de facturación para un Contrato Enterprise cuando la organización firma este contrato específico para utilizar Azure. Los administradores de la cuenta de facturación de un Contrato Enterprise utilizan departamentos y ámbitos de facturación de cuentas para administrar y organizar la facturación de la organización.

- **Contrato de cliente de Microsoft**: Se crea una cuenta de facturación para un contrato de cliente de Microsoft cuando la organización firma este contrato específico para utilizar Azure. Los administradores de la cuenta de facturación de un contrato de cliente de Microsoft utilizan perfiles de facturación y ámbitos de facturación de las secciones de factura para administrar y organizar la facturación de la organización. Para obtener más información, consulte [Empezar a trabajar con la cuenta de facturación para un contrato de cliente de Microsoft](billing-mca-overview.md).

## <a name="change-billing-account-view-in-the-azure-portal"></a>Cambio de la vista de la cuenta de facturación en Azure Portal

Un ámbito es un nodo dentro de una cuenta de facturación que los usuarios utilizan para ver y administrar la facturación. Por ejemplo, las cuentas de facturación para Microsoft Online Services Program solo tienen un ámbito, la cuenta de facturación, mientras que las cuentas de facturación para Contrato Enterprise tienen los ámbitos de cuenta de facturación, departamento y cuenta. De forma predeterminada, si tiene acceso a varios ámbitos de facturación, uno de ellos se muestra en la página Administración de costos + facturación de Azure. Para cambiar el ámbito, siga estas instrucciones:

1. Inicie sesión en el [Azure Portal](https://portal.azure.com).

2. Busque en **Administración de costos + facturación**.

   ![Captura de pantalla que muestra la búsqueda en Azure Portal](./media/billing-view-all-accounts/billing-search-cost-management-billing.png)

3. Seleccione **Todos los ámbitos de facturación** en el lado izquierdo.

   ![Captura de pantalla que muestra todos los ámbitos de facturación](./media/billing-view-all-accounts/billing-list-of-accounts.png)

4. Seleccione un ámbito para ver los detalles.
