# <a name="create-an-additional-subscription-in-the-azure-portal"></a>Creación de una suscripción adicional en Azure Portal

Puede crear suscripciones adicionales para su cuenta en Azure. Puede que quiera una suscripción adicional para evitar alcanzar los límites de suscripción, para crear entornos independientes para facturación y seguridad o para aislar los datos por motivos de cumplimiento.

Si quiere crear suscripciones de Azure bajo el Contrato Enterprise de su organización, debe tener el rol de propietario de la cuenta en su organización.

Si aún no tiene una suscripción a Azure, cree una [cuenta gratuita](https://azure.microsoft.com/free).

## <a name="create-an-additional-azure-subscription"></a>Creación de una suscripción de Azure adicional

1. Inicie sesión en el [Azure Portal](https://portal.azure.com).
1. Busque **Suscripciones**.

   ![Captura de pantalla que muestra la búsqueda de suscripciones en el portal](./media/billing-create-subscription/billing-search-subscription-portal.png)

1. Seleccione **Agregar**.

   ![Captura de pantalla que muestra el botón Agregar en la vista Suscripciones](./media/billing-create-subscription/subscription-add.png)

También puede crear suscripciones adicionales mediante programación. Para más información, consulte [Creación de suscripciones de Azure Enterprise mediante programación](../azure-resource-manager/programmatically-create-subscription.md).
