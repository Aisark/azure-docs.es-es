# <a name="quickstart-prepare-your-network-for-azure-defender-for-iot-deployment"></a>Inicio rápido: Preparación de la red para la implementación de Azure Defender para IoT

## <a name="choose-your-hardware"></a>Selección del hardware
Azure Defender para IoT admite varias aplicaciones certificadas, entre las que se incluyen:
- Dispositivos preconfigurados certificados en los que el software ya está instalado.
- Aplicaciones certificadas no configuradas en las que puede descargar e instalar software de sensor.
- Especificaciones del dispositivo de la consola de administración local. La consola de administración local no está disponible como dispositivo preconfigurado. 

Para más información, consulte la [Guía de especificaciones de hardware](https://aka.ms/AzureDefenderforIoTBareMetalAppliance).


## <a name="install-components"></a>Instalar componentes
Después de adquirir los componentes de hardware y descargar el paquete de software del portal de Azure Defender para IoT, puede instalar el software de componentes. Para más información, consulte la [Guía de instalación](https://aka.ms/AzureDefenderforIoTInstallSensorISO).
