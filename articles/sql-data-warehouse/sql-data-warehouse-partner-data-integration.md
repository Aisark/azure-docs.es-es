# <a name="sql-data-warehouse-data-integration-partners"></a>Socios de integración de datos de SQL Data Warehouse
Para crear la solución de almacenamiento de datos, elija entre una amplia variedad de herramientas líderes del sector. En este artículo se destacan las empresas asociadas de Microsoft con soluciones de integración de datos oficiales compatibles con Azure SQL Data Warehouse.

## <a name="data-integration-partners"></a>Asociados de integración de datos
| Asociado | DESCRIPCIÓN | Vínculo del sitio web o el producto |
| ------- | ----------- | -------------------- |
| ![Alooma][7] |**Alooma**<br> Alooma es una solución ETL que permite a los equipos de datos integrar, enriquecer y transmitir datos desde varios silos de datos a SQL Data Warehouse, y todo en tiempo real. |[Página de producto][alooma_website] |
| ![Alteryx][1] |**Alteryx**<br> Alteryx Designer proporciona un flujo de trabajo repetible para análisis de datos de autoservicio que conduce a información más detallada en horas, no en las semanas típicas de los enfoques tradicionales. Alteryx Designer ayuda a los analistas de datos combinando la preparación de datos, la fusión de datos y el análisis (predictivo, estadístico y espacial) en la misma interfaz de usuario intuitiva. |[Página de producto][alteryx_designer_website]<br>[Azure Marketplace][alteryx_marketplace]<br>|
| ![Attunity][2] |**Attunity (CloudBeam)**<br>Attunity CloudBeam proporciona una solución automatizada para cargar datos en SQL Data Warehouse. Simplifica la carga por lotes y la replicación incremental de datos de varios orígenes: SQL Server, Oracle, DB2, Sybase, MySQL y muchos más. |[Página de producto][attunity_cloudbeam_website]<br>[Azure Marketplace][attunity_marketplace]<br> |
| ![Denodo][3] |**Denodo**<br>Denodo proporciona acceso en tiempo real a los datos a través de los diversos orígenes de datos de una organización. Usa la virtualización de datos para unir los datos a través de muchos orígenes sin necesidad de una replicación. Asimismo, ofrece un amplio acceso a los datos estructurados y no estructurados que se guardan en orígenes empresariales, de macrodatos y en la nube, tanto en lotes como en tiempo real.|[Página de producto][denodo_website]<br>[Azure Marketplace][denodo_marketplace]<br> |
| ![Fivetran][8] |**Fivetran**<br>Fivetran le ayuda a centralizar los datos de diferentes orígenes. Cuenta con un producto de canalización de datos de configuración que prácticamente no hace falta mantener, con una lista creciente de conectores integrados a todas los orígenes de datos populares. La configuración tarda cinco minutos en realizarse después que se complete la autenticación en los orígenes de datos y el almacén de datos de destino.|[Página de producto][fivetran_website]<br> |
| ![Informatica][4] |**1.Informatica Cloud Services para Azure**<br> Informatica Cloud ofrece la mejor solución de su clase para las capacidades de gestión, integración y migración de datos de autoservicio. Los clientes pueden realizar importaciones de forma rápida y confiable, y exportar petabytes de datos a Azure desde una variedad de orígenes. Informatica Cloud Services for Azure proporciona una conectividad nativa de alto rendimiento y gran volumen con Azure SQL Data Warehouse, SQL Database, Blob Storage, Data Lake Store y Azure Cosmos DB. <br><br> **2.Informatica PowerCenter** PowerCenter es una plataforma de integración de datos, basada en metadatos, que inicia y acelera los proyectos de integración de datos para entregar datos al negocio más rápido que la codificación manual. Sirve como la base para sus inversiones de integración de datos. |**Informatica Cloud Services para Azure**<br>[Página de producto][informatica_Cloud_Services_website]<br>[Azure Marketplace][informatica_Cloud_Services_marketplace]<br><br> **Informatica PowerCenter**<br>[Página de producto][informatica_PowerCenter_website]<br> [Azure Marketplace][informatica_PowerCenter_byol]<br>|
| ![Information Builders][5] |**Information Builders (Administración de datos Omni-Gen)**<br>La plataforma de administración de datos Omni-Gen de Information Builder proporciona soluciones para la integración de datos, al calidad de esos datos y la administración de datos maestros. Asimismo, facilita el acceso, el movimiento y la combinación de todos los datos, independientemente del formato, la ubicación, el volumen o la latencia.|[Página de producto][omnifocus_website]<br> [Azure Marketplace][omnigen_marketplace] |
| ![Qubole][9] |**Qubole**<br>Qubole proporciona una plataforma nativa en la nube que permite a los usuarios realizar cargas de trabajo ETL, de análisis y de IA/ML. Admite una variedad de motores de código abierto: Apache Spark, TensorFlow, Presto, Airflow, Hadoop, Hive y más. Asimismo, proporciona herramientas para el usuario final fáciles de usar en el procesamiento de datos; por ejemplo, proporciona herramientas de consulta SQL y cuadernos y paneles que aprovechan los potentes motores de código abierto.|[Página de producto][qubole_website]<br> [Azure Marketplace][qubole_marketplace] |
| ![Segment][10] |**Segmento**<br>Segment es una solución de análisis y administración de datos que le ayudará a comprender los datos de los clientes que provienen de diversos orígenes. Le permite conectar sus datos a más de 200 herramientas para que pueda crear mejores decisiones, productos y experiencias. Segment transformará y cargará varios orígenes de datos en su almacén usando para ello los conectores de datos integrados.|[Página de producto][segment_website]<br> |
| ![Skyvia][11] |**Skyvia (integración de datos)**<br>La integración de datos de Skyvia proporciona un asistente que automatiza las importaciones de datos. Este asistente le permite migrar datos entre una variedad de orígenes: CRM, base de datos de aplicaciones, archivos CSV y mucho más. |[Página de producto][segment_website]<br> |
| ![SnapLogic][6] |**SnapLogic**<br>La plataforma SnapLogic permite a los clientes transferir rápidamente datos desde y hacia Microsoft Azure SQL Data Warehouse. Ofrece la capacidad de integrar cientos de aplicaciones, servicios y escenarios de IoT en una única solución.|[Página de producto][snaplogic_website]<br>[Azure Marketplace][snaplogic_marketplace]<br> |
| ![Talend][12] |**Talend Cloud**<br>Talend Cloud es una plataforma de integración de datos empresariales para conectar, acceder y transformar cualquier información en la nube o en las instalaciones. Se trata de una plataforma como servicio (iPaaS) de integración que proporciona una amplia conectividad, calidad de datos incorporada y soporte nativo para las últimas tecnologías de nube y macrodatos. |[Página de producto][talend_website]<br> [Azure Marketplace][talend_marketplace] |
| ![Trifacta][13] |**Trifacta Wrangler**<br> Trifacta permite que los usuarios y las organizaciones a exploren y combinen datos diversos para el análisis. Trifacta Wrangler está diseñado para manejar cargas de trabajo de manipulación de datos que necesitan soportar datos a gran escala y un gran número de usuarios finales.|[Página de producto][trifacta_website]<br> [Azure Marketplace][trifacta_marketplace] |
| ![WhereScape][14] |**Wherescape RED**<br> WhereScape RED es un entorno de desarrollo integrado que proporciona a los equipos herramientas de automatización para agilizar los flujos de trabajo de extracción, transformación y carga de datos. El entorno de desarrollo integrado proporciona procedimientos recomendados y código nativo optimizado para destinos de datos populares. Use WhereScape RED para reducir el tiempo para desarrollar, implementar y usar su infraestructura de datos.|[Página de producto][wherescape_website]<br> [Azure Marketplace][wherescape_marketplace] |
