# <a name="azure-synapse-analytics-business-intelligence-partners"></a>Asociados de inteligencia empresarial de Azure Synapse Analytics
Para crear la solución de almacenamiento de datos completa con SQL Analytics en Azure Synapse, puede elegir entre una amplia variedad de herramientas líderes del sector.  En este artículo se destacan las empresas asociadas de Microsoft con soluciones de inteligencia empresarial oficiales (BI) compatibles con Azure SQL Data Warehouse.
