En este inicio rápido, se usa la CLI de Voz desde la línea de comandos para reconocer la voz grabada en un archivo de audio y generar una transcripción a texto. La CLI de Voz es fácil de usar para realizar tareas comunes de reconocimiento, como transcribir conversaciones. Después de una configuración única, la CLI de Voz le permite transcribir el audio a texto de forma interactiva con un micrófono o desde archivos mediante un script por lotes.
