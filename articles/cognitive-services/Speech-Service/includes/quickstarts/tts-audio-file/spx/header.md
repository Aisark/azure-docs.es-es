En este inicio rápido, se va a usar la CLI de Voz desde la línea de comandos para convertir texto a voz almacenada en un archivo de audio. El servicio de conversión de texto a voz proporciona numerosas opciones para voces sintetizadas que puede encontrar en [Compatibilidad con idiomas de conversión de texto a voz](../../../../language-support.md#text-to-speech). Después de una configuración única, la CLI de Voz le permite sintetizar voz a partir de texto mediante comandos de la línea de comandos.
