# <a name="how-to-find-endpoints-and-access-keys-for-the-text-analytics-cognitive-service"></a>Búsqueda de los puntos de conexión y las claves de acceso para Text Analytics en Cognitive Services

Cuando se registra en Text Analytics, obtiene una clave de acceso personalizada única para su suscripción. Esta clave es necesaria en cada llamada a Text Analytics API. Si no se ha [registrado](text-analytics-how-to-signup.md), hágalo ahora para obtener la clave. 

Si necesita ayuda para buscar la clave o para determinar si su suscripción ya tiene Text Analytics, use las instrucciones siguientes para obtener la información necesaria. 

## <a name="find-your-service-endpoint-and-access-key"></a>Búsqueda del punto de conexión de servicio y la clave de acceso

1. Inicie sesión en el [portal de Azure](https://portal.azure.com).

2. En el panel de navegación izquierdo, seleccione **Todos los servicios**.

3. En Filtro, escriba *Cognitive Services*. Si su suscripción tiene Text Analytics, aparece en la lista, con un Tipo de API de **Text Analytics API**.

4. Haga clic en el vínculo para abrir la hoja del servicio. Ahora puede obtener una clave desde **Recursos > Claves** o haciendo clic en **Mostrar claves de acceso** en el panel de Essentials. El punto de conexión también puede verse en el panel de Essentials.
