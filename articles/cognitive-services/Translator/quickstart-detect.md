# <a name="quickstart-use-the-translator-to-detect-text-language"></a>Inicio rápido: Uso de Traductor para detectar el idioma del texto

En este inicio rápido, obtendrá información sobre cómo detectar el idioma del texto proporcionado mediante la API REST de Traductor.

En esta guía de inicio rápido, se requiere una [cuenta de Azure Cognitive Services](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) con un recurso de Traductor.
