# <a name="quickstart-use-the-translator-to-translate-text"></a>Inicio rápido: Uso de Traductor para traducir texto

En este inicio rápido, aprenderá a traducir una cadena de texto de inglés a alemán, italiano, japonés y tailandés mediante la API REST de Traductor.

En esta guía de inicio rápido, se requiere una [cuenta de Azure Cognitive Services](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) con un recurso de Traductor.
