# <a name="knowledge-exploration-service"></a>Servicio Knowledge Exploration

Este es el servicio Microsoft Knowledge Exploration Service.  Knowledge Exploration Service (KES) ofrece una manera rápida y eficaz de agregar búsqueda interactiva y refinamiento a las aplicaciones.  Con KES, puede compilar un índice comprimido a partir de datos estructurados, elaborar una gramática que interprete consultas en lenguaje natural y proporcionar formulación de consultas interactiva con sugerencias de finalización automática.

Si desea un tutorial de ejemplo sobre cómo usar KES para crear una interfaz de búsqueda interactiva para un dominio de publicaciones académicas, consulte la [introducción](GettingStarted.md).

![AcademicAutoComplete](AutoComplete.png)

KES se ha movido de la versión preliminar de Cognitive Services a Cognitive Services Labs. La nueva página de inicio del proyecto es: [https://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration](https://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration).
