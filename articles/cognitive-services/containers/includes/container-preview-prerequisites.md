## <a name="prerequisites"></a>Prerequisites

* Use una suscripción de Azure. Si no tiene una suscripción a Azure, cree una [cuenta gratuita][free-account] antes de empezar.
* Instale la [CLI de Azure][azure-cli] (az).
* [Motor de docker][docker-engine] y se asegura de que la CLI de Docker funciona en una ventana de consola.

[free-account]: https://azure.microsoft.com/free
[azure-cli]: https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest
[docker-engine]: https://www.docker.com/products/docker-engine
