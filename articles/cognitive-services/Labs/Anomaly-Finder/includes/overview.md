## <a name="overview"></a>Información general
Anomaly Finder API permite supervisar los datos con el tiempo y detectar anomalías con aprendizaje automático, que se adapta a sus datos únicos aplicando automáticamente el modelo estadístico correcto. Permite a las organizaciones mantener la calidad de los datos, proporcionar servicios de confianza, identificar incidentes empresariales y refinar el enfoque de negocio sin abordar los problemas difíciles de identificación manual de datos anómalos.

### <a name="version-information"></a>Información de versión
*Versión*: 1.0.0

### <a name="uri-scheme"></a>Esquema de URI
*Host*: labsportalppe.azure-api.net  
*BasePath*: /anomalyfinder/v1.0  
*Esquemas*: HTTPS
