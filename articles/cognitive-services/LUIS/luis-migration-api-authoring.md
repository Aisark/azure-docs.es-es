# <a name="authoring-api-changes"></a>Cambios en la API de creación 

Las API de creación se actualizaron, en el caso de la versión V3, para permitir la [descomposición del modelo con entidades de aprendizaje automático](luis-concept-data-extraction.md). 

Puede crear en las API V2 o V3 versión preliminar. La misma aplicación también puede optar por usar las respuestas del punto de conexión de predicción V2 o V3. No es necesario que use la misma versión de la API para la creación y la respuesta del punto de conexión de predicción.

El portal de LUIS usa las API de creación V3 exclusivamente.

## <a name="next-steps"></a>Pasos siguientes

Revise [cambios en el punto de conexión de predicción](luis-migration-api-v3.md) para la API V3.
