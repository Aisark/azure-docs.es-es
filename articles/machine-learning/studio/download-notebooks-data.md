# <a name="download-notebookspreview-data"></a>Descargar datos de cuadernos (versión preliminar)

La característica Cuadernos (versión preliminar) se quitará el *13 de abril de 2020*. Después de esa fecha, la pestaña Cuadernos (versión preliminar) desaparecerá y no se podrán restaurar los datos de los cuadernos. Descargue los datos de sus cuadernos antes del 13 de abril de 2020.

En este artículo se proporcionan instrucciones paso a paso sobre cómo descargar los datos de Cuadernos (versión preliminar).

1. Vaya a la pestaña **Cuadernos** de Azure Machine Learning Studio (clásico).
    ![Vaya a la pestaña Cuadernos.](./media/download-notebooks/notebooks-list.png)

1. Seleccione el cuaderno que quiere descargar. Se abrirá en Jupyter.
    ![Seleccionar un cuaderno](./media/download-notebooks/select-notebook.png) No se pueden descargar varios cuadernos a la vez.

1. Vaya a **Archivos** -> **Descargar como** y seleccione una opción de formato.
    ![Opciones de descarga de cuadernos](./media/download-notebooks/download-options.PNG)

1. Después de seleccionar una opción se empezará a descargar el archivo del cuaderno. Omita los errores 500 o 502 que aparezcan; no afectarán a la descarga.
