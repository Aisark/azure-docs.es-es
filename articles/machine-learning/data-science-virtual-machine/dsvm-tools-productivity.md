# <a name="productivity-tools-on-the-data-science-virtual-machine"></a>Herramientas de productividad en Data Science Virtual Machine

Además de las herramientas de ciencia de datos y programación, DSVM contiene herramientas de productividad que le ayudarán a capturar y compartir información con sus compañeros. Office 365 ProPlus es la experiencia de Office más productiva y más segura para empresas, y permite que los equipos trabajen juntos sin problemas desde cualquier lugar y en cualquier momento. Con Power BI Desktop puede pasar de los datos a las conclusiones y a la acción. Además, el explorador Microsoft Edge es un explorador web moderno, rápido y seguro.
