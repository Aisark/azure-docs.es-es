# <a name="azure-resource-manager-templates-for-management-features"></a>Plantillas de Azure Resource Manager para las características de administración

En la tabla siguiente se incluyen vínculos a plantillas de Azure Resource Manager para las características que proporciona Resource Manager.

| | |
|-|-|
|**Asignaciones de roles**||
| [Asignación de roles para el grupo de recursos](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-resourcegroup)| Asigna un rol integrado a un usuario para un grupo de recursos existente. |
| [Asignación de roles para la máquina virtual existente](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-virtualmachine)| Asigna un rol integrado a un usuario para una máquina virtual existente. |

| [Asignación de roles para una suscripción de Azure](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-deployments/subscription-role-assignment)| Asigna un rol a un usuario para una suscripción de Azure. |
|**Definición de roles**||
| [Creación de definiciones de roles personalizadas](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-deployments/create-role-def)| Crea una nueva definición de roles en una suscripción de Azure. |
|**Bloqueo de recursos**||
| [Bloqueo de grupo de recursos](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-deployments/create-rg-lock-role-assignment)| Crea un grupo de recursos y aplica un bloqueo **DoNotDelete** al grupo de recursos. Asigna el rol de colaborador a un usuario. |
