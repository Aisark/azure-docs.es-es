# <a name="array-functions-azure-cosmos-db"></a>Funciones de matriz (Azure Cosmos DB)
[!INCLUDE[appliesto-sql-api](includes/appliesto-sql-api.md)]

Las funciones de matriz le permiten realizar operaciones en matrices en Azure Cosmos DB.

## <a name="functions"></a>Functions

Las siguientes funciones escalares realizan una operación en un valor de entrada de matriz y devuelven un valor numérico, booleano o de matriz:

* [ARRAY_CONCAT](sql-query-array-concat.md)
* [ARRAY_CONTAINS](sql-query-array-contains.md)
* [ARRAY_LENGTH](sql-query-array-length.md)
* [ARRAY_SLICE](sql-query-array-slice.md)
