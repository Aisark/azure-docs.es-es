1. En la sección **Configuración** de la página de la red virtual, seleccione **Servidores DNS** para abrir la página **Servidores DNS**.

2. En la página **Servidores DNS**, rellene los valores siguientes:

   - **Servidores DNS**: seleccione **Personalizada**.

   - **Agregar servidor DNS**: Introduzca la dirección IP del servidor DNS que quiere usar para la resolución de nombres.

3. Una vez que agregue los servidores DNS, seleccione **Guardar**.

    ![Especificación de un servidor DNS](./media/vpn-gateway-add-dns-rm-portal-include/specify-dns-server.png "Specify a DNS Server")
