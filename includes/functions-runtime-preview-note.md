> La versión preliminar de Azure Functions Runtime 2 solo admite la versión 1.x del runtime de Azure Functions. Esta característica de vista previa no se está actualizando para admitir la versión 2.x de runtime y no las actualizaciones futuras se planea. Si necesita para hospedar el tiempo de ejecución de Azure Functions fuera de Azure, considere la posibilidad de [mediante un contenedor de Linux cliente](../articles/azure-functions/functions-create-function-linux-custom-image.md).
