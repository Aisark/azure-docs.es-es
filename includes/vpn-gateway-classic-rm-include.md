Azure actualmente funciona con dos modelos de implementación: Resource Manager y clásico. Los dos modelos no son totalmente compatibles entre sí. Antes de empezar, es preciso saber el modelo en que se desea trabajar. Para obtener información sobre los modelos de implementación, consulte [Modelos de implementación de Azure](../articles/resource-manager-deployment-model.md). Si es la primera vez que usa Azure, se recomienda usar el modelo de implementación de Resource Manager.
