Microsoft 365 se creó para que se pueda acceder a él de forma segura y confiable a través de Internet. Por ello, se recomienda utilizar ExpressRoute en escenarios concretos. Para más información sobre cómo usar ExpressRoute para acceder a Microsoft 365, visite [Azure ExpressRoute para Microsoft 365](/microsoft-365/enterprise/azure-expressroute).
