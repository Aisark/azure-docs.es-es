## <a name="set-up-authentication"></a>Configuración de la autenticación

Para obtener acceso al servicio, deberá proporcionar una clave de cuenta, el token de acceso o el token de autenticación de Azure Active Directory.

### <a name="account-keys"></a>Claves de cuenta

Las claves de cuenta son una credencial que permite que la aplicación para autenticarse con el servicio de Azure espacial delimitadores. La finalidad de las claves de cuenta es que le ayudarán a empezar a trabajar rápidamente. Especialmente durante la fase de desarrollo de integración de la aplicación con Azure espacial delimitadores. Por lo tanto, puede usar las claves de cuenta mediante la inserción en las aplicaciones cliente durante el desarrollo. A medida que avanza más allá de desarrollo, se recomienda encarecidamente mover a un mecanismo de autenticación es el nivel de producción, compatible con Tokens de acceso o la autenticación de usuario de Azure Active Directory. Para obtener una clave de cuenta para el desarrollo, visite su cuenta de Azure espacial delimitadores y vaya a la pestaña "Claves".
