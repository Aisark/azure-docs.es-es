## <a name="delete-anchors"></a>Eliminación de delimitadores

Para eliminar un anclaje espacial en la nube, use el método `DeleteAnchor()`. La eliminación de los anclajes que ya no se usan es una práctica recomendada que debe incluirse en las primeras fases del proceso de desarrollo para mantener limpios los recursos de Azure.
