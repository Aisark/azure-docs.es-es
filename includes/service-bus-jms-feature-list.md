En la tabla siguiente se enumeran las características de **Java Message Service (JMS)** que se admiten actualmente y que Azure Service Bus no admite.


| Características | Estado |
|---|---|
| Colas   | Compatible |
| Temas   | Compatible |
| Colas temporales | Compatible |
| Temas temporales | Compatible |
| Selectores de mensajes | Compatible |
| Exploradores de colas | Compatible |
| Suscripciones duraderas compartidas | Compatible|
| Suscripciones duraderas no compartidas | Compatible |
| Suscripciones no duraderas compartidas | Compatible |
| Suscripciones no duraderas no compartidas | Compatible |
| Transacciones distribuidas | **No compatible** |
| Término duradero | **No compatible** |
