1. Vaya a Azure Portal.
2. Agregar un nuevo recurso, seleccione el **crear un recurso** botón en el portal de Azure.
3. Seleccione **Function App sin servidor**.
4. Escriba toda la información en la configuración de creación y establezca su **Plan de hospedaje** a **Plan de App Service**:
    
    ![Crear plan de la aplicación de función](./media/functions-premium-create/create-function-app-plan.png)  

    1. Seleccione un nivel de plan (EP) Premium para el plan de servicio. 
        ![Seleccione el plan premium](./media/functions-premium-create/hosting-plan.png)  

    1. Validar la configuración de plan completo.  
        ![Plan de servicio de aplicación finalizada](./media/functions-premium-create/create-app-service-plan.png)  
    
1. Validar la configuración de aplicación y el plan y seleccione **crear**.

    ![Plan de servicio de aplicación finalizada](./media/functions-premium-create/create-function-app.png)
