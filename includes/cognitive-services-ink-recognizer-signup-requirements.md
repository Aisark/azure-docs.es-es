Debe tener un [cuenta de API de Cognitive Services](../articles/cognitive-services/cognitive-services-apis-create-account.md) con acceso a la API de reconocimiento de tinta. Si no tiene una suscripción de Azure, puede [crear una cuenta](https://azure.microsoft.com/try/cognitive-services/) gratuita. Puede obtener la clave de suscripción en [Azure Portal](../articles/cognitive-services/cognitive-services-apis-create-account.md#access-your-resource) después de crear la cuenta, o en el [sitio web de Azure](https://azure.microsoft.com/try/cognitive-services/my-apis) después de activar una evaluación gratuita.
