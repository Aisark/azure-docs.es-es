## <a name="versions-of-the-azure-cli"></a>Versiones de la CLI de Azure

Hay dos versiones de la interfaz de la línea de comandos (CLI) de Azure disponibles actualmente:

* [CLI de Azure](../articles/storage/common/storage-azure-cli.md): La versión actual de la CLI de Azure para su uso con el modelo de implementación de Resource Manager.
* [Azure CLI clásica](../articles/storage/common/storage-azure-cli-nodejs.md): Una versión anterior de la CLI de Azure, destinado a usarse solo con el modelo de implementación clásica.
