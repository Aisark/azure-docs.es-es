Para trabajar con Azure Digital Twins en este artículo, antes es preciso **configurar una instancia de Azure Digital Twins** y los permisos necesarios para usarla. Si ya tiene una instancia de Azure Digital Twins configurada del trabajo anterior, puede usarla.

De lo contrario, siga las instrucciones que se indican en [Configuración de una instancia y autenticación](../articles/digital-twins/how-to-set-up-instance-portal.md). Las instrucciones también contienen pasos para comprobar que ha completado cada paso correctamente y está listo para pasar a usar la nueva instancia.

Después de configurar la instancia de Azure Digital Twins, tome nota de los siguientes valores que necesitará para conectarse a la instancia más adelante:
* **_nombre de host_** de la instancia. Puede encontrar este valor en [Instrucciones](../articles/digital-twins/how-to-set-up-instance-portal.md#verify-success-and-collect-important-values) en Azure Portal.
* la **suscripción de Azure** que usó para crear la instancia (su nombre o identificador servirán). Para ver la suscripción en la que se encuentra una instancia de Azure Digital Twins, vea la misma página de *información general* de la instancia en [Azure Portal](https://portal.azure.com).
