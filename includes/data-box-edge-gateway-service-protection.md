- Para obtener acceso al servicio de borde del cuadro de datos, su organización debe tener una suscripción de contrato Enterprise (EA) o proveedor de soluciones en la nube (CSP). Para obtener más información, consulte [registrarse para obtener una suscripción de Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Dado que este servicio de administración se hospeda en Azure, se está protegido por las características de seguridad de Azure. Para obtener más información acerca de las características de seguridad proporcionadas por Azure, vaya a la [Microsoft Azure Trust Center](https://azure.microsoft.com/support/trust-center/security/).
- Para las operaciones de administración de SDK, puede obtener la clave de cifrado para el recurso en **las propiedades del dispositivo**. Puede ver la clave de cifrado solo si tiene permisos para el recurso de Graph API.
