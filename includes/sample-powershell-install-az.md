Si es necesario, instale el módulo Azure PowerShell con las instrucciones de la [guía de Azure PowerShell](/powershell/azure/) y ejecute `Connect-AzAccount` para crear una conexión con Azure. Además, debe haber una clave pública SSH denominada "`id_rsa.pub`" en el directorio .ssh del perfil de usuario.
