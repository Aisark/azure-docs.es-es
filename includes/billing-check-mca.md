Compruebe el tipo de contrato para determinar si tienen acceso a una cuenta de facturación para un contrato de cliente de Microsoft.

1. Inicie sesión en el [Azure Portal]( http://portal.azure.com).

2. Busque en **Administración de costos + facturación**.

   ![Captura de pantalla que muestra la búsqueda en Azure Portal](./media/billing-check-mca/billing-search-cost-management-billing.png)

3. Si tiene acceso a una sola cuenta de facturación, seleccione **Propiedades** desde el lado izquierdo. Tiene acceso a una cuenta de facturación para un contrato de cliente de Microsoft si el tipo de cuenta de facturación es **Contrato de cliente de Microsoft**.

    ![Captura de pantalla que muestra el contrato de cliente de Microsoft en la página de propiedades](./media/billing-check-mca/billing-mca-property.png)

4. Si tiene acceso a varias cuentas de facturación, compruebe el tipo en la columna de cuenta de facturación. Tiene acceso a una cuenta de facturación para un contrato de cliente de Microsoft si el tipo de cualquiera de las cuentas de facturación es **Contrato de cliente de Microsoft**.

    ![Captura de pantalla que muestra el contrato de cliente de Microsoft en la página de lista de cuenta de facturación](./media/billing-check-mca/billing-mca-in-the-list.png)
