## <a name="intent-recognition-from-a-file-with-events"></a>Reconocimiento de intención desde un archivo mediante eventos

El siguiente código muestra cómo reconocer la intención en el idioma predeterminado (en-US) de manera continua. El código permite el acceso a información adicional, como los resultados intermedios. La entrada procede de un archivo de audio. El formato compatible es WAV/PCM de un solo canal (mono) con una frecuencia de muestreo de 16 KHz.
