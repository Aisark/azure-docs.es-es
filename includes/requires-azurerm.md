> [!IMPORTANT]
>
> El uso de esta característica de Azure desde PowerShell requiere que tenga el módulo `AzureRM` instalado. Se trata de un módulo anterior que solo está disponible para Windows PowerShell 5.x y que recibe actualizaciones importantes, pero ya no obtiene nuevas características. Si tiene el módulo `Az` instalado, también deberá instalar `AzureRM`. Para instalar `AzureRM`:
> 
> 1. Deshabilite los alias de AzureRM en `Az` (si están habilitados) con `Disable-AzAlias`. Ejecutar este comando sin `Az` instalado o con los alias deshabilitados es inofensivo.
> 2. Siga las [instrucciones de instalación de AzureRM más recientes](/powershell/azure/install-az-ps). Incluso si tiene `AzureRM` instalado, debe actualizar a la versión más reciente.
