Cuando se usa Microsoft Azure Traffic Manager para equilibrar la carga del tráfico que va al sitio web de Azure, se puede tener acceso a ese sitio mediante el nombre de dominio **\*.trafficmanager.net** asignado por Azure. También puede asociar un nombre de dominio personalizado, como www\.contoso.com, con su sitio Web con el fin de proporcionar un nombre de dominio más reconocible para los usuarios.
