También puede instalar y ejecutar los cmdlets de Azure PowerShell localmente en el equipo. Los cmdlets de PowerShell se actualizan con frecuencia. Si no ha instalado la versión más reciente, los valores especificados en las instrucciones pueden dar lugar a errores. Para buscar las versiones de Azure PowerShell instaladas en el equipo, use el cmdlet `Get-Module -ListAvailable Az`. Para instalar la actualización, vea [Instalación del módulo de Azure PowerShell](/powershell/azure/install-az-ps).
