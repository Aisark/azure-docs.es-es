## <a name="prerequisite-install-the-azure-cli"></a>Requisito previo: Instalación de la CLI de Azure

Para seguir los pasos de este artículo, [instale la CLI de Azure](/cli/azure/install-azure-cli) e [inicie sesión en Azure](/cli/azure/authenticate-azure-cli). 

> [!NOTE]
> Si no tiene una cuenta de Azure, necesitará una. Regístrese para [obtener una prueba gratuita aquí](../articles/active-directory/fundamentals/sign-up-organization.md).
> Además, para seguir el tutorial necesita disponer de [jq](https://stedolan.github.io/jq/) o de alguna otra herramienta de análisis o biblioteca de JSON instaladas.
