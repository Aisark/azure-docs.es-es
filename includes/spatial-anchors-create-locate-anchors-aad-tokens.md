### <a name="azure-active-directory-authentication"></a>Autenticación con Azure Active Directory

Delimitadores espacial Azure también permite que las aplicaciones se autentiquen con tokens de Azure AD (Active Directory) del usuario. Por ejemplo, puede usar los tokens de Azure AD para integrar con Azure espacial anclajes. Si una empresa mantiene a los usuarios de Azure AD, puede proporcionar un token de usuario de Azure AD en el SDK de Azure espacial delimitadores. Si lo hace, podrá autenticarse directamente en el servicio de Azure espacial delimitadores para una cuenta que forme parte del mismo inquilino de Azure AD.
