### <a name="azure-active-directory-authentication"></a>Autenticación con Azure Active Directory

Delimitadores espacial Azure también permite que las aplicaciones se autentiquen con tokens de Azure Active Directory del usuario. Por ejemplo, puede usar tokens de AAD para integrarse con ASA. Si una empresa mantiene los usuarios de AAD, puede proporcionar un token AAD del usuario en el SDK de Azure espacial delimitadores. Si lo hace, podrá autenticarse directamente en el servicio ASA para una cuenta que forme parte del mismo inquilino AAD.
