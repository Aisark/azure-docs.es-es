El establecimiento de un nombre de dominio personalizado en una aplicación web integrada en Traffic Manager solo está disponible para el plan de tarifa **Estándar**.  

Para más información sobre los planes de tarifa de App Service, incluido cómo cambiar el plan de tarifa de la aplicación, consulte [Escalado vertical de aplicaciones en Azure](../articles/app-service/manage-scale-up.md).
