Azure Data Lake Storage Gen1 es un repositorio a gran escala de nivel empresarial para cargas de trabajo de análisis de macrodatos. Puede usar Data Lake Storage Gen1 para capturar los datos de cualquier tamaño, tipo y velocidad de ingesta en un único lugar para realizar análisis exploratorios y operativos. No hay ningún límite en la cantidad de datos que puede almacenar en una cuenta de Data Lake Storage Gen1.

| **Recurso** | **Límite predeterminado** | **Comentarios** |
| --- | --- | --- |
| Número máximo de cuentas de Data Lake Storage Gen1, por suscripción por región |10 | Para solicitar un aumento de este límite, póngase en contacto con soporte técnico. |
| Número máximo de ACL de acceso, por archivo o carpeta |32 | Se trata de un límite máximo. Usar grupos para administrar el acceso con menos entradas. |
| Número máximo de ACL predeterminadas, por archivo o carpeta |32 | Se trata de un límite máximo. Usar grupos para administrar el acceso con menos entradas. |
