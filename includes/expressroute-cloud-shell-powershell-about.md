En este artículo se usan cmdlets de PowerShell. Para ejecutar los cmdlets, puede usar Azure Cloud Shell, un entorno de shell interactivo hospedadas en Azure y usar a través del explorador. Azure Cloud Shell viene con los cmdlets de PowerShell de Azure preinstalados.

Para ejecutar cualquier código incluido en este artículo en Azure Cloud Shell, abra una sesión de Cloud Shell, use la **copia** en un bloque de código para copiar el código y péguelo en la sesión de Cloud Shell con botón __Ctrl + Mayús + V__ en Windows y Linux, o __Cmd + Mayús + V__ en macOS. Texto pegado no se ejecuta automáticamente, por tanto, presione **ENTRAR** para ejecutar el código.

Puede iniciar Azure Cloud Shell con:

|  |   |
|-----------------------------------------------|---|
| Seleccione **Probarlo** en la esquina superior derecha de un bloque de código. Esto __no__ copiar automáticamente el texto en Cloud Shell. | ![Ejemplo de Probarlo para Azure Cloud Shell](./media/cloud-shell-try-it/cli-try-it.png) |
| Abra [shell.azure.com](https://shell.azure.com) en el explorador. | [![Botón Iniciar Cloud Shell](./media/cloud-shell-try-it/launchcloudshell.png)](https://shell.azure.com) |
| Seleccione el botón **Cloud Shell** en el menú de la esquina superior derecha de [Azure Portal](https://portal.azure.com). | ![Botón Cloud Shell en Azure Portal](./media/cloud-shell-try-it/cloud-shell-menu.png) |

Si no desea usar Azure Cloud Shell, instale Azure PowerShell localmente en su lugar. Asegúrese de instalar la versión más reciente del módulo Azure PowerShell Az para obtener funcionalidad actualizada. Para más información, vea [Instalar Azure PowerShell](/powershell/azure/overview).
