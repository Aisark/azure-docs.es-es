Estos procedimientos recomendados, tenga en cuenta:

- Se recomienda almacenar todas las contraseñas en un lugar seguro para que no tenga que restablecer una contraseña si se le olvida. El servicio de administración no puede recuperar contraseñas existentes. Solo puede restablecerlas a través del portal de Azure. Si restablece una contraseña, asegúrese de notificar a todos los usuarios antes de hacerlo.
- La interfaz de Windows PowerShell del dispositivo puede tener acceso de forma remota a través de HTTP. Como práctica recomendada de seguridad, debe utilizar HTTP solo en redes de confianza.
- Asegúrese de que las contraseñas de dispositivos segura y bien protegidos. Siga el [prácticas recomendadas de contraseñas](https://docs.microsoft.com/azure/security/azure-security-identity-management-best-practices#enable-password-management).
