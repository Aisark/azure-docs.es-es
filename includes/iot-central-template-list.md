**Plantillas de aplicación**

| Plantilla                 | Nombre        | Descripción |
| ------------------------ | ----------- | ----------- |
| iotc-pnp-preview@1.0.0   | Aplicación personalizada | Permite crear una aplicación vacía para que pueda rellenarla con sus propias plantillas de dispositivo y dispositivos. |
| iotc-default@1.0.0       | Aplicación personalizada (heredada) | Permite crear una aplicación heredada vacía para que pueda rellenarla con sus propias plantillas de dispositivo y dispositivos.
| iotc-condition@1.0.0     | Análisis en la tienda: supervisión del estado | Permite crear una aplicación para conectar y supervisar un entorno de almacén. |
| iotc-consumption@1.0.0   | Supervisión del consumo de agua | Permite crear una aplicación para supervisar y controlar el flujo de agua. |
| iotc-distribution@1.0.0  | Centro de distribución digital | Permite crear una aplicación para mejorar la eficacia de la salida de productos del almacén mediante la digitalización de recursos y acciones clave. |
| iotc-inventory@1.0.0     | Administración inteligente del inventario | Permite crear una aplicación para automatizar la recepción, el movimiento de productos, el recuento de ciclos y el seguimiento. |
| iotc-logistics@1.0.0     | Logística conectada | Permite crear una aplicación para realizar un seguimiento de su envío en tiempo real por aire, mar y tierra con supervisión del estado y la ubicación. |
| iotc-meter@1.0.0         | Análisis inteligente de medidores | Permite crear una aplicación para supervisar el consumo de energía y el estado de la red e identificar tendencias para mejorar la asistencia al cliente y la administración de medidores inteligentes.  |
| iotc-mfc@1.0.0           | Centro de micrologística | Permite crear una aplicación para conectar y administrar digitalmente un centro logístico totalmente automatizado. |
| iotc-patient@1.0.0       | Supervisión continua de pacientes | Permite crear una aplicación que amplía la atención al paciente, reduce las readmisiones y administra las enfermedades. |
| iotc-power@1.0.0         | Supervisión de energía solar | Permite crear una aplicación para supervisar el estado del panel solar y las tendencias de generación de energía. |
| iotc-quality@1.0.0       | Supervisión de la calidad del agua | Permite crear una aplicación para supervisar digitalmente la calidad del agua. |
| iotc-store@1.0.0         | Análisis en el almacén: finalización de la compra | Permite crear una aplicación para supervisar y administrar el flujo de finalización de compra en su tienda. |
| iotc-waste@1.0.0         | Administración de residuos conectada | Permite crear una aplicación para supervisar contenedores de residuos y operadores de campo de distribución. |
