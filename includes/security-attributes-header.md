Un atributo de seguridad es una calidad o una función de un servicio de Azure que contribuye a la capacidad del servicio para prevenir, detectar y responder a las vulnerabilidades de seguridad.

Los atributos de seguridad se clasifican en las siguientes categorías:
* Prevención
* Segmentación de red
* Detección
* Compatibilidad con la administración de identidades y acceso (IAM)
* Pista de auditoría
* Controles de acceso (si se utilizan)
* Administración de la configuración (si se utiliza)

En todas las categorías, especificamos si el atributo se utiliza o no (sí/no). Para algunos servicios, un atributo puede no ser aplicable y se muestra como no disponible. También puede incluirse una nota o un vínculo para aportar más información sobre un atributo.
