> [!IMPORTANT]
> Hay dos módulos de PowerShell que se usan para interactuar con Service Fabric. [Azure PowerShell](/powershell/azure/install-az-ps) se utiliza para administrar los recursos de Azure, como un clúster de Service Fabric hospedado en Azure. El [SDK de Azure Service Fabric](../articles/service-fabric/service-fabric-get-started.md) se utiliza para conectarse directamente al clúster de Service Fabric (independientemente de dónde esté hospedado) y administrar el clúster, las aplicaciones y los servicios.
