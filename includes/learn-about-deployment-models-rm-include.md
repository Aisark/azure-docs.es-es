> [!NOTE]
> Azure tiene dos modelos de implementación diferentes que puede utilizar para crear recursos y trabajar con ellos: [Azure Resource Manager y clásico](../articles/azure-resource-manager/management/deployment-models.md). Este artículo trata sobre el modelo de implementación del Administrador de recursos. Se recomienda usar el modelo de implementación de Resource Manager para las nuevas implementaciones, en lugar del modelo de implementación clásica.
