> [!NOTE]
> Azure tiene dos modelos de implementación diferentes que puede usar para crear y trabajar con recursos: [Azure Resource Manager y clásico](../articles/azure-resource-manager/resource-manager-deployment-model.md). En este artículo se explica el uso del modelo de implementación de Resource Manager. Se recomienda el modelo de implementación de Resource Manager para nuevas implementaciones en lugar del modelo de implementación clásica.
