Para que la aplicación lógica tenga acceso a cualquier servicio, es preciso crear una *conexión* entre ambos. Si no la ha creado antes, al agregar un desencadenador o una acción de dicho servicio a su aplicación lógica se le pide información de la conexión. El diseñador de aplicaciones lógicas proporciona una manera sencilla de crear esta conexión directamente desde cualquier aplicación lógica.
