Azure Data Share permite a las organizaciones compartir datos de forma simple y segura con sus clientes y asociados.

| **Recurso** | **Límite** |
| --- | --- |
| Número máximo de recursos de Data Share por suscripción de Azure |50 |
| Número máximo de recursos compartidos enviados por recurso de Data Share |100 |
| Número máximo de recursos compartidos recibidos por recurso de Data Share |100 |
| Número máximo de invitaciones por recurso compartido enviado |100 |
| Número máximo de suscripciones a recursos compartidos por recurso compartido enviado |100 |
| Número máximo de conjuntos de datos por recurso compartido |100 |
| Número máximo de programaciones de instantáneas por recurso compartido |1 |
