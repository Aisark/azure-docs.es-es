> [!NOTE]  
> La zona horaria predeterminada que se usa para ejecutar las expresiones CRON es la Hora universal coordinada (UTC). Para que la expresión CRON se ejecute según otra zona horaria, cree una configuración de aplicación para la aplicación de función denominada WEBSITE_TIME_ZONE. Para más información, consulte [Zonas horarias de NCRONTAB](../articles/azure-functions/functions-bindings-timer.md#ncrontab-time-zones).
