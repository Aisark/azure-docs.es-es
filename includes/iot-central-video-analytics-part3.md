## <a name="version-the-device-template"></a>Creación de versiones de la plantilla de dispositivo

La aplicación análisis de vídeo: detección de movimiento y objetos incluye la plantilla de dispositivo **LVA Edge Gateway**. Para usar el archivo de manifiesto de implementación actualizado con esta plantilla de dispositivo, debe crear y publicar una nueva versión de la plantilla.

Para crear una nueva versión de la plantilla:

1. En la aplicación IoT Central, vaya a **Plantillas de dispositivo** y seleccione la plantilla de dispositivo **LVA Edge Gateway**.

1. Seleccione **Versión** para crear una plantilla denominada **LVA Edge Gateway v2** y, después, seleccione **Crear**.

La nueva plantilla de dispositivo **LVA Edge Gateway v2** ahora está en la lista de plantillas de dispositivo.
