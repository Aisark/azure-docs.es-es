Si no está familiarizado con Cognitive Services o con Azure, cree una [clave temporal](https://azure.microsoft.com/try/cognitive-services/). Si tiene una cuenta de Azure, cree un nuevo recurso en el portal. Las consultas de punto de conexión de Language Understanding admiten dos tipos de recursos de Azure: El recurso de **Language Understanding** o el de **Cognitive Services**. Necesitará el nombre del recurso cuando [asigne](../articles/cognitive-services/luis/luis-how-to-azure-subscription.md) la clave a la aplicación.
