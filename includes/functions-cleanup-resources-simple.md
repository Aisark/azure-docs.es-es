## <a name="clean-up-resources"></a>Limpieza de recursos

Cuando termine de trabajar con los recursos creados en este artículo, use el siguiente comando para eliminarlos de la suscripción:

```azurecli
az group delete --name myResourceGroup
```

Cuando se le solicite, escriba `y`. Este comando elimina el grupo de recursos `myResourceGroup` y todos los recursos que pertenecen a él.
