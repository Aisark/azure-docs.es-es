> [!IMPORTANT]
> En este artículo se incluye contenido que requiere la CLI de Azure clásica. La versión actual de la CLI de Azure no es compatible con las características que se describen en este artículo, por lo que se requiere la CLI clásica. 
> 
> La CLI clásica se puede instalar junto con la CLI de Azure moderna, pero es aconsejable usar la CLI Azure para todos los nuevos scripts e implementaciones.
> Para instalar la CLI clásica, consulte [Instalación de la CLI clásica de Azure](/cli/azure/install-classic-cli). 
> Para instalar la versión más reciente de la CLI, consulte [Instalación de la CLI de Azure](/cli/azure/install-azure-cli).
