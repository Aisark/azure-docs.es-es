Para evitar que se incluyan archivos innecesarios en la instantánea, cree un archivo ignore (`.gitignore` o `.amlignore`) en el directorio. Agregue los archivos y directorios que se van a excluir a este archivo. Para obtener más información sobre la sintaxis que se debe usar dentro de este archivo, vea [sintaxis y patrones](https://git-scm.com/docs/gitignore) para `.gitignore`. El archivo `.amlignore` usa la misma sintaxis. _Si ambos archivos existen, el archivo `.amlignore` tiene prioridad._
