## <a name="more-information-about-bots"></a>Más información sobre los bots

Para obtener más información sobre cómo usar este servicio con los bots, comience con los siguientes recursos:

|Recurso|Propósito|
|--|--|
|[Azure Bot Service](https://dev.botframework.com/)|Azure Bot Service proporciona un servicio web hospedado en la nube completo con un punto de conexión de bot. El servicio usa [Bot Framework](https://github.com/Microsoft/botframework), que está disponible en varios idiomas.|
|[Bot Framework](https://github.com/Microsoft/botframework)|Microsoft Bot Framework es una plataforma completa para crear experiencias de IA de conversación de nivel empresarial.|
|[Bot Framework Emulator](https://github.com/Microsoft/botframework#Bot-Framework-Emulator)|Bot Framework Emulator es una aplicación de escritorio multiplataforma que permite que los desarrolladores de bots prueben y depuren los bots mediante el SDK de Bot Framework. Puede usar Bot Framework Emulator para probar los bots que se ejecutan localmente en la máquina o para conectarse a los bots que se ejecutan de forma remota.|
|[Herramientas de bot](https://github.com/Microsoft/botbuilder-tools)|Bot Framework Tools es una colección de herramientas de línea de comandos multiplataforma diseñada para abarcar el flujo de trabajo de desarrollo de bots de un extremo a otro. |
|[Ejemplos de Bot Builder](https://github.com/Microsoft/BotBuilder-Samples)|Se han diseñado ejemplos de bot totalmente desarrollados para ilustrar escenarios que deberá implementar para crear bots excelentes.|
